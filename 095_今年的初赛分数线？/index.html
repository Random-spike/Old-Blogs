今年的初赛分数线？ 新文
2023-10-22 - /095_今年的初赛分数线？/

今天是考完初赛的日子，家长们也是几多欢喜几多愁，各家情况都不一样。

省流：

1- 小周老师退出江湖？

2- J组分数线多少

3- 没考好怎么办

////

////

1- 小周老师退出江湖？

都说了小周老师押不中题，就退出江湖。

然而，目前不是小周老师押没押中的问题，而是押中多少的问题。

简单来说，就是押中了一大堆。

细致来说，我们就截图说明好了。

1- 这虽然考的八进制，但八进制和二进制完全不分家的好么。

2- 妥妥二叉树变种，二叉树不理解，三叉树绝无可能做出来；二叉树理解，三叉树也一样，这里还顺手考了个完全三叉树的概念

3- 高精度还真不是什么复杂的东西，根本上，还是要理解二进制加法，加法的算法理解了，10进制竖式加法的肌肉记忆也就变成逻辑记忆了。

4- X缀表达式，在这里是后缀表达式，题型还挺新颖，一点点拆，算不上模板，但肯定是可以做

5- 都说了，二进制和八进制没有明显区别。这里难道不应该想一下为什么不是二进制和二十八进制？为什么一定是八进制或者十六进制？这东西不应该30秒做完么。

6- 我说没说过，哈夫曼编码是其中一个？

7- 经典X缀遍历问题，没有比这种题更经典的了，画树啊我说

8- 如果有谁说0和1两个值的bit和二进制无关的话，那我觉得真的也不要学奥赛了

所以，15个选择题小题，我押中了8个。

我这都不用提完善程序里面的二分查找和DP。

这加一起得50分了。

所以小周老师有可能押不中而退出江湖么？

没可能。

都说了，我既然说这句话，就一定有必胜的把握。

而且，这只是我押的二进制相关的知识点。

有没有这样一种可能，我上奥赛课的时候，讲的内容，是这个的10倍？

不夸张的说，这套考题里面每一道题的知识点，我都讲过，不止两遍。

2- J组分数线多少

我赛后看了题之后的第一感觉，就是太容易。

这样容易的题，分数线肯定会高，但高多少，这就有讲究了。

高多少，取决于给多少机位

首先，我知道满天飞的都是初赛评估小程序。在这些评测里面，都有个分数线的评估。

我也知道，评估中，目前北京的分数线，是在72.5到73.5之间。

但这东西不准。

第一，谁也不知道北京最后给多少机位，所以只能通过参加评测的学生群体来猜。

第二，去参加评测的同学，可能还没有20%，那这个通过学生群体来估算，以及比对去年学生群体，那没有机位数量，那就一定不准。

但不管怎么不准，是有一点肯定的，那就是分数线一定比去年高。

会不会上70，这，是个问题。

首先这个难度，真的是有点容易了。从计算上来说，肯定是要上70的。

但是70，这是个心理上的坎。最近7、8年，北京初赛分数线，就从来没有上过70。这次要真是一咬牙上了70，那势必，所有学校都要开始卷初赛。

这是个巨大的资源的浪费。

如果北京是个信息奥赛绝对强省，有无限的资源，那行，就算是卷到90分，也可以继续，一直卷到95为止。

只是，北京到目前来说，顶多算是个信息奥赛中强省。

不是说北京孩子弱，而是因为北京的机会太多，没必要这么卷信息奥赛也能一本。

不信？

您看看清北在北京招多少学生就知道。

您再看看北京多少985，211就更清楚了。

所以，既然是中强省，既然学生们连训练复赛的时间都挤出来弄这个初赛，那，是不是应该不要煽风点火，见到风就是雨地往上抬分数线呢？

难道没看到今年都有四岁的小宝来卷初赛了吗？

[手拿准考证的小娃]

你弄个浙江一样的分数线，心不会痛嘛？

所以，从这个角度来看，北京的分数线70+，是个需要妥善处理的棘手问题。

从另一个角度来说，浙江，就没什么悬念。

我第一时间看完题就估了85，都不用看数据。

浙江这个分数线，和各个平台是一致的。也不太可能变化。

这应该属于地域差异的问题。

所以，小周老师估计北京的分数线是68。

最后是不是这个分，就要看博弈关系了。

3- 没考好怎么办

没考好，这要看具体情况。

如果您教练不教提高组，只教普及组，甚至只陪初赛刷题，那就要警觉了。

得观察周围的情况，

1- 如果有考得好的同学，那就继续加油，就当和小教练一起成长了，您交钱，他成长。

您娃能不能成长这得看苗子。

2- 如果周围都砸锅了，一锅端，那就赶紧换机构吧。没啥好说的。
