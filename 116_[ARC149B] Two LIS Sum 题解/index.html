题目大意：  link

给定两个序列 AAA，BBB。

可以任意的交换 AiA_iA​i​​ 与 Ai+1A_{i+1}A​i+1​​ 但同时要交换 BiB_iB​i​​ 与 Bi+1B_{i+1}B​i+1​​ 交换。

解题思路：  link

首先猜结论，把 AAA 排成 111，222，333，⋯\cdots⋯，NNN，然后求出被改过的的 BBB 的 LIS\text{LIS}LIS。

证明结论，放两张官方题解的图便于理解。

橙色的代表在该数列的 LIS\text{LIS}LIS 里面，白色则代表不在。


第一种情况，AAA 数列在排序时，把一个对应不在 BBB 数列的 LIS\text{LIS}LIS 序列的数，排序之后，还是不在 LIS\text{LIS}LIS 序列里面，所以这种情况，总 LIS\text{LIS}LIS 长度加一。

第二种情况，AAA 数列在排序时，把一个对应在 BBB 数列的 LIS\text{LIS}LIS 序列的数，排序之后变成不在 LIS\text{LIS}LIS 序列里面，所以这种情况，总 LIS\text{LIS}LIS 长度加零。

第三种情况，AAA 数列在排序时，把一个对应不在 BBB 数列的 LIS\text{LIS}LIS 序列的数，排序之后变成在 LIS\text{LIS}LIS 序列里面，所以这种情况，总 LIS\text{LIS}LIS 长度加二。

第四种情况，AAA 数列在排序时，把一个对应在 BBB 数列的 LIS\text{LIS}LIS 序列的数，排序之后还是在 LIS\text{LIS}LIS 序列里面，所以这种情况，总 LIS\text{LIS}LIS 长度加一。

可以看到，在每次操作中，LIS\text{LIS}LIS 总和至少不会减小，在 AAA 所消除的逆序数对肯定大于 BBB 数列的减小个数，所以即使是最坏情况 LIS\text{LIS}LIS 也会是 AAA 数列所产生的 nnn。
