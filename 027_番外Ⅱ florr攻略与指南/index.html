番外Ⅱ florr攻略与指南 florr
2024-08-17 - /027_番外Ⅱ florr攻略与指南/

往期文章请查看 合集 。

公会  link

公会通常作用是组车队/聊天

组车队一般含有挂机车和farm车，一般相近水平的玩家会组成公会。

公会变化速度非常快，半年前的grim还是顶尖公会，1个月前的star还是世界第二的公会。

目前情况是NF垄断几乎所有高水平玩家。

NPC  link

NPC有两中，trader和Oracle。

trader  link

作用是把卡片换成同等级coin。coin伤害血量和同等级骨相同。

虽然scoin放在沙轮中并不出色，但是scoin在转速1.2s的轮子（地狱轮，scoin轮）中dps是大于usand的。scoin平常farm比几乎大部分u卡都强。（包括地狱粉卡）

trader一般在沙漠左边红区向右下的延伸部分，但是trader会动。在一个服务器几个小时会把trader重刷在原位置。

Oracle  link

Oracle可以将一定数量低等级花瓣换成高等级花瓣，成功率为100%。但是所需花瓣数高于期望值。冷却时间为27min。

Oracle通常会在海洋的通道处游荡，因为海洋的通道并不宽，所以寻找起来难度较小。

Epic 19rare
