更快的速度  link

在有多张 U 卡之后 （大约30张），比较重要的事情是更新配装，比如说跑路装。

跑路装通常由泡泡，light，天线，阴阳构成，这些花瓣至少应该是Mythic稀有度（阴阳除外）

获取召唤流  link

召唤流在很多地方都是挂机获取花瓣的必备物品，比如在下水道，fah，ck。

沙漠是高级召唤物的唯一来源。所有也是为了获取召唤流的必刷之地，一般沙漠的挂机车队会在地图四周，farm队伍会在地图中间。

沙漠的u怪一般伤害都不低，beetle的体伤，蝎子的毒伤都有比较大的威胁。因此在没有比较强的把握下尽量带上一个海绵（再给海绵配一个回血装）。

因为有很多使用沙轮的人来刷沙漠，所以尽量集中伤害打一个怪能保证更多的掉落。

在bgg和stick有一定数目后，可以使用Queen获得的agg进行挂机(当然，肯定进不去一些高级队伍)。

在这段刷沙漠的时间中，可以留心获取 Mygg，现在沙漠会刷出比较多数的Ultra Shinyladybug，因此mygg不再稀有。

海洋综合发展  link

沙轮  link

沙轮现在成为这个版本必须有的东西。

在有了full bgg+stick（其实也可以不满但尽量满），可以更好的在crab kindom刷泡泡，沙子，爪子等非常有用的花瓣。

ck一般挂机点在顶部的分割墙2处，底部一处，一般来说，顶部的效率会更加高（可以获取更多的泡泡），ck最理想的情况便是3个队伍12个人构成的挂机池，效率比较高，理想状态1个小时可以期望出一个Ultra claw。

获取沙轮需要夜以继日的挂机，但获取满Ultra 泡泡更难，在合成泡泡的基础上，可以通过抢Super的方式尝试通过掉落的方式获取。

如果实在难以获取全满的泡泡，3个粉泡泡在平时farm可以满足大部分需要。

Omaze  link

Omaze 本来是一个难度比较大的区域，后来经过削减后难度并不大，怪的构成主要为m+U怪，在遇到相应的怪做好相关的准备。

  • U jellyfish：Urubber（没有U的可以使用M的）。

  • U leech：准备好回血装，回血装的背包应该带battery/lightning

  • U Crab：如果要安全解决可以使用三眼，相信自己的技术可以不携带。

  • U Starfish：携带dandelion。

解决以上问题可以很轻松的2人组队/刷Omaze。但是速度可能有待提升。

O end(trench)  link

O end(trench)是海星区下方的S型衍生区域，里面基本生物构成为U怪。

打发和O maze是大同小异的，但是为了安全和效率，可以组更多人的队伍，比如通过公会的方式。

挂机  link

有了基本的一套Ultra后，接下来的目标应该定在出S。

由于现在抢S难度的陡增和farmUltra的效率的提升，抢S不再是获取Super的唯一途径。

CK挂机点  link

常见的3个挂机位置已经讲了。

  • 掉落物：实用。

  • 效率：一般。

  • 难度：中。（全队至少24bgg）

沙漠挂机点  link

位置在沙漠右下角方块的边角。

  • 掉落物：一般。

  • 效率：一般偏低。

  • 难度：中。（也取决于位置）（全队至少24bgg）

火蚁穴  link

位置在FAH最下面的一格。

  • 掉落物：不实用。

  • 效率：比较高。

  • 难度：中高，（在最中心）（如果要非常安全的挂机需要一个Super召唤）

下水道  link

位置一般处于下水道方块的四个角，队伍很有实力可以位位置。

  • 掉落物：不实用。

  • 效率非常高。

  • 难度：极高。（只存在少量挂机点的情况下必须有Super召唤和多数U树枝才能保证安全的挂机）

注：本文安全的挂机指3小时以上完全没有任何非人为灭队事件。
