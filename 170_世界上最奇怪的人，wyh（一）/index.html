世界上最奇怪的人，wyh（一） 旧文
2023-01-18 - /170_世界上最奇怪的人，wyh（一）/

序言  link

还没网课时，wyh说过我要再一次搞出奇怪巅峰（上一次搞出的震惊全班的奇怪巅峰因为奇奇怪怪的图床倒闭了，所以里面的图片大部分没了，所以这次我要一雪前耻！），不过呢，我相信wyh想不到，这是关于他的特辑啊哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈。

说明：因为篇幅较大，我会放在多篇文章里面，最后会统一起来的，大家先一篇一篇看，或者不看也行，点个赞哈。

问笔记群  link

为了证明我是大好人，没有对wyh有偏见和针对，我打算先介绍一下更为奇怪的问笔寄群问笔记群。嗯太奇怪了。

肯定和以前一样，先介绍一下历史。

事实上，有两个问笔记群。

第一个问笔记群  link

第一个问笔记群，是zms在4月1日创办的。当时十分正气，在各种奇奇怪怪的群中脱颖而出，成为了规模很大的一个群。这个群的创办时间不算早，但是业务范围准确单一，问笔记。

我是在5月泥巴巴拉进来的，那时候这个群已经开始娱乐化。（我之所以进来是因为这头像tm是我…）里面各种奇奇怪怪的东西都出现了。我还成功引进了表情包。所以当时问笔记群是表情包和某些人爱情故事为主流的。有些人死了，但他还活着。有些人还活着，但他还活着！

第一次革命  link

在第一个问笔记群，第一次革命无疑动摇了其曾坚定不移的根基。而这次革命的领导者，便是我们伟大坚强的wyh。当时，猥琐群主zms愈发猥琐，随意禁言而且一禁言就是1,2天，到处撤回。管理员们如同florr官群一般腐败不堪，两者之间有小小的矛盾就会直接把那个人踢掉（我被踢过数次），还总是乱改名字，改成*** 爱 *** 。真是十分邪恶！！！！！！

于是，wyh率领着我们开始展示无_产_阶_级的怒火。对暴君开战，正面威慑把群主架空，新建群聊。

发动方式：先软开战让我们革命党拥有群管理员权限，接下来直接踢掉所有人正面威慑，结束。

革命在人民群众的支持下（大约15人）大约半天时间，我们获得全面胜利！获得胜利之后，zms被迫退位，转让十分正气的录播局（这是我们投票投出来的）当时florr党开始发展，于是录播局还立下条约说自己少聊游戏，管理员大部分都是投出来的，是革命中，做出莫大贡献的成员。

第二次革命的祸根  link

好景不长，在第一次革命后的一周，环境相对宽松，我们可以自由的讨论问题，刷屏，kecp，聊游戏，发奇怪的投票和作业。管理员撤回消息的事时有发生，但没有如此频繁。

同时，由于第一次革命是问笔记群的名声跌入谷底，所以，各种各样的群开始出现，几乎每个人都建立了，规模最大我觉得还是维也纳的内卷群，大概13人罢，之后维也纳不允许任何人加入。（13人在当时算很大的，问笔记群只有15，现在纪录是24）里面风气也很正。

前群主zms也貌似又改过自新的状态。所以我们都很高兴。

但是这已经埋下了祸根。

第二次革命的爆发  link

祸根是什么？祸根在于问笔记群里面的交流过于自由，而例如维也纳的内卷群，交流内容很正气单一，（当然也有电动群，斗图群等等），由于播菊试图阻止问笔记群没有规章制度这一现状，再一次开始禁言，踢人。后来，录播局可能害怕自己是下一个zms，或者实在管不下去，也可能两种情况都有。彻底放开了，里面全是游戏内容，各种稀奇古怪的游戏都拿出来，而且斗图刷屏情况时时发生，所以我试图阻止这一情况，同时发动第二次革命。

这次革命并没有成功，原因很多。

第一点：第一次革命机遇主义者过多，第一次革命，很多人，比如邪恶猥琐摸鱼渣，都是想蹭个管理员的，第二次革命时，他们都没有动力了，安于现状不思进取，这是他们的过错。

第二点：这是我们总体方针的过错，一句话解释就是太刚了，我们当时决策是剔除一切杂质，只留革命骨干，思想纯洁的，没有布尔乔亚安于现状的。所以，我们在执行这一决策时，树敌过多，把原本是我们的人搞到敌人那边去了。那时候我并没有读过毛选等书籍，或者说不是一名合格的革命者不知道什么是全世界无产阶级联合起来，也不知道什么是团结一切可团结力量，wyh等人可能没看到我之前发的消息，就被我们踢了。所以成为了我们的敌人，我们双拳难敌四手，这是我们的决策问题。

第三点：第三段就很烦，zzq带头瞎搞，zzq一阵乱七八糟的人身攻击，一阵霍霍之后，zzq把所有人都踢了，然后录播局被渣渣怂恿解散，然后录播局宣布解体，（因为距离第一次群主转让没有30天，不能转让群主）

总而言之，第二次革命的结果产生原因很多，但是捅上最后一刀的还是天天摸鱼猥琐渣。

革命后  link

因为问笔记群的彻底解体，让各个小群迎来了发展巅峰，全班大概又20-30此类小群干什么的都有，我熟悉的比如斗图群，刷屏群，表情包群，还有各种各样奇奇怪怪的群借着问笔记群解题的养料，都发展起来。

当时场面一度混乱。

约莫是解体后一周的一节英语课，我问zms，你愿意重建问笔记群吗，他得出肯定回答，于是我们着手总结问题，5月18，记住这关键的日子。

1.群主猥琐到处禁言

2.有反动势力搞事情

3.管理制度不够严密

4.群风过于散漫随意

5.第一次革命威信扫地，别的群太多了。

我们列出可加入人的名单，

8，15,16,17,2,0，21,23,26,27,29,30,32,33，35.

总共14人。

14人中拒绝的，很多。

平时非常大好人的27都说;确实不行，留个纪念吧…

我记得王子奇说了些什么，但是他的消息记录全部诡异消失了，所以不放了。

（看不清点这里点这里）

就连我们的骨干成员17,31都产生了些许动摇。甚至zms也觉得问笔记群将成为历史。

何况猥琐摸鱼渣，上一个问笔记群最大罪人，建立了新的“问笔记群”，还在群名后加了“正版”两个字都没人改…,天天聊游戏，天天斗图，不免有些空虚吧…

我们的小小群，在各种冲击下，夹缝求生，节节败退，人最少时，只有7个人，别的人，我们怎么留也留不住，8，17,23，26,27,30,35，只有我们7个。

当时，真的心灰意冷，问笔记群成为历史，只在瞬间。

当日，有个校庆活动，我把何嘎嘎不停地拉，试图和何嘎嘎说一些奇奇怪怪的东西，比如何嘎嘎喜爱的zxy。何嘎嘎则不停地说一阵后退出，退出。约莫是第4次退出后，我试图做最后的尝试，这次，何嘎嘎终于没有走了，这貌似是何嘎嘎除班委群外加的第一个奇怪的群，这也是问笔记群的转折点。

发展之路  link

发展之路，很难，但是我们已经挺过去了成立之初的灾难，一夜间，开始彻底逆袭，我们抓着何嘎嘎提供的救命稻草，开始重生。

没人看好我们，但我们看好我们

挺过第一个寒冬之后，我们靠着何嘎嘎和泥巴巴强大的魅力相助，收复了很多问笔记群的骨干成员，第一次扩建之后，我们群里面有8,14,15,16,17,20,23,24,25,26,27,30,31,33,35,37.

后来，在六一时，我们的群已经规模很大了，但仅仅是与zzq猥琐摸鱼群相当。

再后来，六月中旬，录播局来了，录播局的到来代表着我们终于彻底收复了我们失去的骨干成员，新的问笔记群也打败了众多奇奇怪怪的小群，小群中解散的解散，安静的安静下来了。第二个问笔记群再次占据全班主导地位。渣渣器的猥琐摸鱼群也都是一天几乎都没人说话了。

最后的胜利  link

最后，猥琐摸鱼渣渣器终于顶不住了，他的群已经安静无比，他也加入了我们第二个问笔记群！

这就是问笔记群的一切历史。

至今，合法加入过同时因老妈袭击而不是主动退出的人，有这些。

1,3,8，11,13（春菊专门拉过来搞波ju的），14,15，16,17,20,21,22,23,24，25,26,27,29,30,31,32,33,34,35,36，37.

粗算一下，26个人，全班大约2/3了。

谁能想到它经历过的困难？

危难时刻  link

12月29日中午。有人告密了，我被姜菊抓去审问，有关问笔记群的组织和聊天记录，试图让我告密，可惜，经过10分钟的审问，什么也没审问到。接下来，我用了一个伟大坚强c++电脑病毒搞死了电脑，我家长就无法偷袭电脑查找聊天记录了。

接下来，老姜召集班委和群主张铭祥审问，在审问过后，依旧什么也没有审问出来，当日下午，问笔记群紧急解散，只剩下我，zms和wcx。

问笔记群解散后，伟大坚强！王禹蘅！成立了一个奇怪的群，虽然我不知道那个群以前干嘛，但是至少知道那暂时成为聊天群，但是十分不方便。

12/3，12/4我们逐渐拉回骨干成员，问笔记群重生。

回到中心  link

我相信，到这时候，大部分读者可能都被带偏到问笔记群上，为什么和奇奇怪怪猥琐不堪的wyh有关系呢？我们在现在，最好有截图证明的，时代背景，不都和问笔记群相关么？再者，是时候纪录它了4.1~大约5月15，5.20，至今。了
