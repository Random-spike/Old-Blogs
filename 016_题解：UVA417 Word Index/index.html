题解：UVA417 Word Index 题解
2024-09-06 - /016_题解：UVA417 Word Index/

UVA 417  link

题意：根据题目要求排定保证严格升序的字符串的编码。

根据题意，可以看出题目要求解决以下问题：

  • 排列出所有符合要求的字符串。
  • 在排列出字符串的基础上排出每个字符串的编号。

对于这两个问题使用如下解决方案：

  • 使用 555 重循环，在循环时考虑空位的情况，其他情况要求保证严格单调递增。
  • 根据题目性质，在正确的排列符合要求的字符串后，即可按照字符串的排列顺序依次给该字符串给定一个编号。

最后使用 map 储存所有的答案。
