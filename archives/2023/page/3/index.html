想要了解C++？来，带你一起 新文
2023-10-21 - /097_想要了解C++？来，带你一起/

前两天在一个信奥教练群里面，看到了这么一条求助信息：

学生现在是五年级，想要了解下C++

主要目的如下

1，认识什么是C++

2，C++和其他的编程语言有什么区别

3，C++具体可以完成什么项目或工作

4，C++学习后可以获得什么（例如比赛，考试等）

我就觉得这套问题挺逗的：这些问题好像挺具体特接地气特诚恳，但实际上，这套求助者总结出来的问题，堪称花拳绣腿空气拳，完美避开所有重要的点。

所以我给的答案是这样的：

1- 国内中小学生奥赛用编程语言

2- 奥赛用

3- 奥赛

4- 奥赛

这还真是一个认真的回答，一点儿没瞎写。

下面就逐一来说明一下

1-没必要认识什么是C++。

你认识C++干啥？去研究计算机语言的命名史？还是去开发游戏引擎？难不成是要去做个让大家翘首以待的计算机操作系统比如能够取代Windows的Doors系统？

显然根本不是。

上面这些内容，连臭名昭著的CSP-J/S的初赛都不考。

初赛顶多考个C和C++谁是面向对象语言，根本不需要知道C++实际上的名字是C with Classes。

所以不需要认识C++。

你只需要知道C++是国内中小学生考信奥需要用的编程语言就行了。

以前还有个Pascal，现在已经全面被C++取代了。

2- C++和别的编程语言的真正区别，你知道了也没用。

比如，C++和Java有什么区别？让我来随便列几个：

1- C++支持指针，Java不能操作指针

2- C++支持多继承，而Java不支持多重继承，但有接口

3- C++面向对象做的不彻底，Java 是完全面向对象的语言，只保留class取消了 C/C++ 中的struct和union

4- Java自动进行无用内存回收操作，不再需要程序员进行手动删除，而 C++ 中必须由程序释放内存资源

5- Java不支持操作符重载，操作符重载则被认为是 C++ 的突出特征。

6- goto语句是 C 和 C++ 的历史遗迹，Java不提供 goto 语句，这使程序更简洁易读。

7- Java不支持 C++ 中的自动强制类型转换，如果需要，必须由程序显式进行强制类型转换。

看懂了么？

显然没看懂对不对？

不管是你是五年级的同学，还是五年级同学的打算让娃学编程的家长。

不是这个专业毕业的职业程序员，看都看不懂。

看不懂就对了。

而且还不考。

所以，C++和其他编程语言有什么区别？

那就是：

C++是奥赛用语言；其他语言不是。

这就是真正的区别。

当然如果一定要抬杠，说USACO能用Java还能用Python呢！

那升级一下回答：

C++快。

剩下的，关于为什么C++快的理由，涉及到CPU、寄存器、机器码以及汇编语言的这些事，你需要知道么？

不需要。

所以，就三个字：

奥赛用。

没了。

3- C++具体可以完成什么项目和学生没关系

能干的事儿多了。

和奥赛生有关的，没有。

就算是你全省第一，当学生的时候，就管当学生的事儿就行了。

你当学生就是要去清华的，我说错了没？

所以别说什么科技创新发明AI算法识别猫脸鲸鱼表情的事情，现在就聊奥赛，不聊别的。

不是么？

所以，这次，就俩字儿。

奥赛。

没了。

4-C++学习后就能考信奥了，这理由够充分不？

麻省理工发明的Scratch语言学习之后能获得什么？

麻省理工的录取通知书吗？

不是，对不对？

连谷歌都在使用的Python语言，学习之后能获得什么？

谷歌的offer吗？

显然也不是，是不是？

C++不学没法考奥赛；学了之后能考奥赛。

就这。

所以，依然是俩字儿：

奥赛。

结语：

如果大家有选择，我觉得去山里骑个马，摘个果子，游个泳，划个船，比什么都好。学什么劳什子奥赛？学什么C++？控制什么指针？解什么图论？

然而。

然而，中考越来越难了。

看着这两年双过来的孩子们面对中考数学压轴题瑟瑟发抖的时候，做家长的，有没有想好怎么给他们多铺一条路？

这看起来好像是风马牛不相及的两件事，其实，这是一件事。

学奥赛，正是因为中考本身的课业太重了。

把全部鸡蛋都放在中考一个篮子里面，一个趔趄，就完蛋。

分出一部分精力，从小学5年级开始，开始准备成为中考前的科技特长生，学好一门手艺，或许能够拯救自己的前途。

所以，真的，学不学随意。

反正前途都是自己的。

噢，不，是孩子自己的。

是信奥的机构，就好好培训学生。

偶尔说个听着有用的，譬如说，“奥赛准备黄金期”，仔细一看主语还都特别混乱：

奥赛和黄金？

期是什么？？

黄金期货？？？

白瞎了“黄金期”这三个字。

今天小周老师就来说说这个“黄金期”。

首先，主语要清晰，“黄金期”，是个主语，前面是要有定语的。这个定语，至少要是“信息奥赛生”。

连在一起，应该是，“信息奥赛生的黄金期”。

但这看起来仍不清晰：

为什么信息奥赛生会有个黄金期？这个黄金期指的是？

所以还需要扩充这个定语变成形容词短语。

信息奥赛生入门的，黄金期。

这看起来就对了：对于打算入门的信息奥赛生来说，有个相关的概念，叫做黄金期。

要想弄明白黄金期，那就要了解在信息奥赛的路上的两件头等大事。

**第一是要早。

第二是要熬。**

所谓的早，就是，如果想要学信奥出成绩，务必要在四年级的暑假开始。

当然二年级也行，这要分教练，也分学生。

对于教练，不管二年级还是初三，能够驾驭语言这是个最基本的要求，光吹牛讲黑话是不够的，要有本事。

对于学生，这就有早慧与否的差异了。但早慧与否，只和低年级有关。

**所以，教练需要的，是能让学生学大学的计算机科学的内容，还学得不烦的能力。

学生需要的，是中文听说读写的能力。**

请注意，不是数学能力，也不是计算机的兴趣，而是中文的听说读写的能力。

那么，我们就把从小学二年级开始的信奥生入门需求，列一列：

二年级的小朋友：
1- 不需要通过初中联赛（CSP-J）的初赛，还太早。他们也根本竞争不过早培生或最普通的初中生。
2- 他们只需要能够让C++跑起来，这就是最直接了当的学习回馈。
3- 虽然小，但比较基本的奥赛编程题在上手之后，也是完全可以做的。
4- 要能看懂题，要能听懂讲解。

三年级的小同学：
1- 仍不把通过CSP-J的初赛作为首要目标，但该学的初赛知识（提高组暨高中联赛的知识），只要能学，就尽量储备。
2- 这个年龄，不管是逻辑能力还是语言理解能力，都有长足进步，所以可以不满足于简单的编程语言本身（在小周老师看来，C++本身，就是简单的）。
3- 不管是不是准备好了，都要开始有对比较复杂的事物的理解能力。
4- 只要能学思维训练（就是奥数啦奥数），就能学信奥
5- 需要有能够在online的信息奥赛中（理解成为没有初赛的初中联赛等级的比赛）拿到奖的能力

四年级的同学：
1- 上述所有要求的客观条件都已经具备
2- 暑假之后就是五年级，从小升初的准备时间上，是最后的练兵机会。
3- 六年级开学的秋天，就必须要拿奖，所以新五的比赛，就和真正的战场没有区别
4- 必须尽全力冲击初中联赛的初赛，就算过不了初赛，也要看看自己的水平差距有多少，在下一年才可以查漏补缺。

五年级的同学：
1- 暑假后开学六年级
2- 暑假后开学六年级
3- 暑假后开学六年级
4- 重要的事情上面已经说了三遍了。暑假不全力拼搏，开学可不是和小学生们抢初中联赛的奖，是和初三生。
5- 恐怖么？还觉得五年级小么？

初中同学（新初一新初二新初三）：

1- 这已经是个全新的时代，锁定的目标是中考；面对的挑战是普一和提一
2- 已经错过最佳入场期了
3- 初中生的分析，在本篇文章中不做讨论。

所以，很显然，这几个年级里面，是存在所谓的信息奥赛生入门的黄金期的。

而这个黄金期，从时间线上来看，有个非常清晰的关键时间节点：

四年级的暑假。

如果（更糟的情况），自己娃，不但是普通娃，而且特别不省心，还没有优秀教练带，那基本上，就是……

完蛋一条路。

唯一解，就是找个好教练。

当然，宇宙之大，无奇不有，也有可能会是下面的这种情况：

第一家长不是计算机科学系毕业的，第二家长对算法复杂度还深度感兴趣，第三家长觉得自己下场子教效果好，那么，我推荐这本书：

（信息学奥赛一本通图片）

福建名师董老师的信奥手册，基本上信奥生人人一本。

绝对不忽悠。就问全天下的提高组奥赛生，有谁没有这本书吧？

都有。

所以，让孩子自学？

有句老话，兵熊熊一个，将熊熊一窝。

套用在教育领域，那就是，教书的人不行，那学生保证行不了。

这就是为什么大家总想要去好的学校，不是因为好的学校桌椅新。

是因为老师好。

所以家长想跨过教练，指导孩子的，请先行品尝一下信息奥赛一本通。

真不贵，去买一本，值。

所以，”信息奥赛生，要早做规划“的”早“字这件事，已经说完了。

第二是要熬。 这个其实也比较好理解，因为初中联赛就是个起点。高中联赛，才是竞技的舞台。

之所以初赛对于小学生有很大的杀伤力，是因为初赛里考的，都是高中联赛的知识。

只不过，初赛的形式看起来是选择题，人畜无害的。

平时蒙对了俩仨的，家长还能觉得自己家娃天赋异禀，除了每年被初赛结果暴击一次之外，娃和家长，都没有矛盾。

这就属于双赢。

总之，当一个奥赛生闯过了初赛关，拿到了普及组一等奖（普一，也就是初中联赛一等奖）之后，下一个目标，就是提一。

提高组一等奖的水平，大概是全北京市，或者一个省，的前130名。

高中初中小学全都算上。

到了提一之后，就要开始熬了。

什么时候能把最前面的大佬们熬到大学，什么时候自己就能有机会了。

但这个“熬”字，从根本上，还是“早”字的延续。

没有“早”，拿到提一的时候，很有可能就已经高二了，基本上没有时间熬别人。

所以，归根到底就是，学信奥，就是现在。

这就是信息奥赛生入门的黄金期，的隐藏意义。

时光荏苒，错过了小学四年级的暑假，可真是留不了级的。

**如果您家的信息奥赛生是初三，那么这篇文章绝不能错过。

如果您家的信息奥赛生是初二，那么文章里面说的事情，就是明年。

如果您家的信息奥塞生是四年级，那这篇文章一样不能错过：再过两年就是小升初，如果不能进早培、元培、十一2+4，那再数三年，就是这个时刻。**

但是如果，您家的奥赛生已经出于各种原因停留在普及组的入门阶段，其未来或者现在的提高组奥赛生的身份，已经进入销毁期、休眠期、自毁期、叛逆期，不一而足，那这篇文章还真不适合您，直接关了比较省时间。

为什么？

因为现在要说的特长生群体，是普及组和提高组的贯穿类同学。可不是简单的普及组会写两句三脚猫的程序的这个能力而已。

而了解北京奥赛圈子的家长，应该知道，能，以及愿意把奥赛生，从零基础历经完全不摸门的新手阶段，培养到普及组，最终一直训练到提高组（S组）一等奖的教练，极少。

小周老师就是其中一个。

虽然培养出来的奥赛生各区很多名校都有，但自身，不隶属于任何学校。

这就直接能证明本篇文章的中立性。

好，进入正文，下面要说几件事情：

**1- 中考科技特长生，往年报考流程
2- 2022几个名校的报考流程（人大附中，首师附，等）
3- 2022年，各个名校的科特生要求？（重点中的重点！）

1- 中考科技特长生，往年报考流程**

所谓百炼方可成钢，这话一点儿不假。

暖洋洋的五月，对于奥赛生来说，就是战场：因为，到底是不是信息奥赛生里面的钢，就要决出高下了。

不管自己是不是钢，都无法改变名校的录取原则：

名校，只要钢。

普通材质的不收。

在具体操作层面，中考前，科技类初三生要参加学校的科技类校考。如果是信息奥赛生，那就相当于一次省赛等级的比赛。

请注意，不是所有的科技特长生都是信息学奥赛生，不过，只有信息学奥赛生的信息奥赛成绩，是最过硬，也最容易测试的。

至于到时候校考，是初中联赛这样的省赛，还是高中联赛的省赛，这分学校。

这里就不具体比较各校的考题了，一句话就是：

**如果自己水平弱一点，就别扎堆考人大附中。

考不上的。**

如果你连提高组的课程就没怎么好好上，那根本就不要考虑，去了也有可能爆零。

不管报什么学校，请注意，报考科技特长生的高中，是排他性的：意思就是，如果你报了首师附，就不能报人大附。

从时间安排上，各个学校在往年也都是同一时间进行。

所以请一定要慎重选择。

如果报考了适合自己的学校，在考试中发挥优秀，通过了校考，那么就会拥有科技特长生的身份。

不过，有科技特长生的身份，和能够走科技特长通道中考被录取是两件事。

来，我放大一下：

**有科技特长生的身份，和能够走科技特长通道中考被录取是两件事。
**
考生仍然需要中考成绩通过被录取学校的科技特长生的标准。

同时，通过报考学校的校考的人数，一定大于录取人数。

这个意思很简单：

在通过校考的奥赛生中，择优录取。

就是字面这个意思。

从往年来看，信息奥赛生通过校考之后，在区里公式名单之后，要各自比拼中考成绩。

所以还是这句话，自己没有提高组一等奖++的能耐，就别报人大附中。

考不上。

当然肯定会有些没啥能耐但特别喜欢忽悠的机构出来说，“啊，家长们请看，去年人大附中的科技特长生的招生简章，明文标记普及组三等奖就能报名”

对，是能报名。

那中考满分660，考个550分也能报人大附中。可以，能报。但有用么？

一个道理。

2- 2022几个重点中学的报考流程
