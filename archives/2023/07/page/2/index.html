我决定还是更新lol

入坑10.2日，约莫是的。暑假入坑的号因为丢了

之前部分我有博客写过了。

第一次记载

这详细记载了我的号在大约10,31日附件发生的事情，此时我的号正 在努力从小傻逼进化成傻逼，最终成功了！

第一次网课  link

大约在12.2日，我出了第一青青速，在12.5,12.6 出青海星青电青天线青蛋。这是第一次网课的一周。

第二次记载

那时候我很勤劳，博客里面还可以翻得出来。

第一次网课后  link

那时候应该还没有绑discord，cp6id太卡学校电脑登不上去，在考虑退游

第三次记载

第二次网课  link

第二次网课刚开始的时候，我的号一天1青，一天2青的速度龟速发展，不过出了青rose等就非常有用。后来，在12.11后运气用完了罢，一张也合不出来，应该是卡在9青。

最后一次记载

接下来，应该就没有任何记载了。

在一周后，我出了第十青青沙子，应该是周五12.16（确信，当天出了青海星青电青毒青蛋，之后日青

在很短的时间内，我集齐了4青蛋，但是看到海洋这么好玩便一天到晚跑到海洋打车队，然后刷刷世界树，一到沙漠高波就歇菜，所以第五个青蛋就死活不出，尽管海洋打上41但合卡太非电到是合的出来但爪子fangs贝壳一个不给出，同时青light青pincer和海洋必备的卡也不出。所以沙漠除了刷世界树的低质量车队，高质量车队1蹭不到2没实力打不动，所以除了海洋以外，沙漠花园基本呈现瘫痪状态，然而海洋也因为dps问题打不上去（之前蹭了一个半粉以及全青电爪车队，但是2傻逼队友（带全粉的和电爪的）带青草妈的打到39被几个青车轮海星螃蟹碾碎了）

总而言之，打到瓶颈了，现在想想fangs，爪不出就算了，应该直接不择手段的刷第五个青蛋，出了之后reset掉跟着sesluke搞粉蛋车队。捅花园老挝，最后重新跑回去打海洋。

又过了一段时间，他妈的傻逼dev给我搞spiral了，粉蛋计划功亏一篑。

spiral时期的前期摸索  link

spiral前期，我基本在蜜蜂区和瓢虫区住了一周，我在1.5日出了2青刺。1。6日出了3青光。我就搬家了。

备注：SES-lightening青卡数量是以每天30青的速度直接力超我每天10青的速度超过我的，但是他只有没有青电1青海星还是9槽，装备比我少一个青电，所以还是逊。

1/9日我大概是打了一天的spiral，嗯，我是傻逼。总共只打了2天spiral

spiral中后期  link

中后期，我就住在沙漠了，天天刷世界树，好不容易有一天刷了2个世界树，一个没捡到lol。

除了打沙漠，我还打海洋刷电，基本提升不大，所以一句话总结：我脑子有问题：spiral时期刷世界树。

注：傻逼SES-lightening在这段时间内青卡数量超过了yuzu1207和rosy panic，他用了2个小时的时间就出了世界树，但是他没有青磁铁，所以还是找我继续打傻逼沙漠。

在spiral的最后一天，我打了一下spiral，被ubabyant和uladybug挤死了，真好玩。

总结一下：spiral时期做到了青卡自由，但距离粉卡还是很远。

maze制前期打maze的日常  link

maze制前期，我确实天天跑去打maze打了大概一周左右，做到了常见青至少种类有了（尽管我合不出来，装备全靠爆）

除了打maze，我还去了anthell1小时刷了9个青wing，去了泡泡区刷了一个小时搞了2泡泡赶路

由于出了70=1的青磁铁，那时候一度400=1成为习惯，大约那时候做到了青卡完全自由，卡片统计截止

挂机下水道  link

挂机下水道计划持续1周多，不管有事没事就用2青泡泡往那里跑，然后挂机。

最初我直接烧掉所有4种花瓣，wing，web，poo，faster，狂升到lv150后
只烧poo，web，其他攒着合卡。

wing炸158个，faster54个，都没出， lv155，寒假结束的定格。
