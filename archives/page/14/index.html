CF154A Hometask 原题链接

题目大意：

给定一个字符串 sss，再给出一些两个字符的子串，求出删除至少多少个字符才能让 sss 中没有这些子串的正序或者逆序形式。

代码实现：

题目要求删去字符来破坏所有的非法子串，那么就需要统计出所有的子串数量，同时因为题目有正/逆序的的子串都不能有的要求，所以尤其注意非法子串连在一起的情况，需要对该子串计算出删去字符数量的最小值。

遍历 sss 字符串，遍历计算出每个子串出现的个数。

其中，注意：

字符串：ababa，子串：ab。

这时候，我们只用删去其中的两个 bbb 就可以在剩余字符串中没有目标子串了。

综上，最小值为每连续子串的最少子串中的字符数量的和。

原题 How Many Knights 传送门

题目大意：

骑士（马）在象棋中走“日”字攻击，可以攻击到不在同一行或者同一列且曼哈顿距离为 3 的格子。

现在给定一个 M×NM\times NM×N 的平面，求出最多放多少个骑士在这个平面里面。

思路分析：

这是一个规律题，经过推算和尝试后不难发现以下结论。

设置变量 ansansans 为可以放马的最多数量。

ans = \begin{cases} M &\text{if } N=1 \\ N &\text{if } M=1 \\ \lfloor\frac{N}{4}\rfloor\times4+2\times\min(N\bmod4,2) &\text{if } M=2 \\ \lfloor\frac{M}{4}\rfloor\times4+2\times\min(M\bmod4,2) &\text{if } N=2 \end{cases}

其余情况下：

ans=⌊N×M+12⌋ans=\lfloor\frac{N\times M+1}{2}\rfloor ans=⌊​2​​N×M+1​​⌋

  1. 如果不需要第二操作就可以让数列 ppp 有序，则输出 000。

  2. 需要第二操作后可以让数列 ppp 有序，则输出 111。

  3. 需要执行 222 次及以上的第二操作后才让数列有序，输出 −1-1−1。

思路分析：

因为数列公差为 111，所以交换的数的差和其编号一样都被 kkk 整除，所以可得到： \lvert p_i-p_j \rvert\bmod k=0。

同时，因为输出的结果只和第二次操作次数有关，因此可以设置变量 cntcntcnt 表示操作次数。

  1. cnt=0cnt=0cnt=0 时，即没有操作过第二操作，输出 000。

  2. cnt=2cnt=2cnt=2 时，因为两个数字交换时， cntcntcnt 会计算两次，故这表示操作过第二操作 111 次，输出 111。

  3. cnt>2cnt>2cnt>2 时，可以理解为做过 111 次以上的第二操作，输出 −1-1−1。
