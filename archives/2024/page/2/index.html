UVA 417

题意：根据题目要求排定保证严格升序的字符串的编码。

根据题意，可以看出题目要求解决以下问题：

  • 排列出所有符合要求的字符串。
  • 在排列出字符串的基础上排出每个字符串的编号。

对于这两个问题使用如下解决方案：

  • 使用 $5$ 重循环，在循环时考虑空位的情况，其他情况要求保证严格单调递增。
  • 根据题目性质，在正确的排列符合要求的字符串后，即可按照字符串的排列顺序依次给该字符串给定一个编号。

最后使用 map 储存所有的答案。

AC code

题解：P11011 「ALFR Round 4」A 点的覆盖 题解
2024-08-30 - /017_题解：P11011 「ALFR Round 4」A 点的覆盖/

题目大意

给定一个矩形，在矩形中有一些散落的点，求出该矩形有多少子矩形覆盖这些点。

题目思路

根据题意，画出一个例子：

红线是在上下左右四个方向距离最靠近矩形的边的点做关于该边的平行线得出来的。

由图可见，如果要画出一个符合要求的矩形，那么矩形的顶点都应该落在打上了标记的地方。

因为如果又一个顶点不在这个区域，那么该子矩形就不能够覆盖在边角的极值点，不符合题目要求。

为了求出答案，我们关注于左上角和右下角打了标记的小矩形（另一个方向和这个是相同的）。

因为顶点必须在小矩形内，因此相应个顶点可以在相应矩形中选取，根据排列组合可以得知结果应该是所有蓝线的长度相乘。

设 $A$ 的左上顶点坐标为 $(x_1,y_1)$，右下顶点坐标为 $(x_2,y_2)$。覆盖点的 $y$ 轴极值为 $y_{\text{min}}$ 和 $y_{\text{max}}$，$x$ 轴极值为 $x_{\text{min}}$ 和 $x_{\text{max}}$。

换成公式就是：

$$\text{ans}=(y_1-y_{\text{max}}+1)\times (y_{\text{min}}-y_2+1)\times (x_2-y_{\text{max}}+1)\times (x_{\text{min}}-x_1+1)$$

题目大意

给定 $n$ 个正整数，要求询问在这 $n$ 个数中有多少数不可以被写成连续的正整数相加的形式。

题目思路

首先分析样例：

  • $3=1+2$
  • $6=1+2+3$
  • $8=1+3+4$

$8$ 不满足要求。

因为题目要求中包含连续的正整数相加，这就可以联想到等差数列求和公式：

$$\frac{(a_1+a_n)\times n}{2}=\displaystyle\sum_{i=1}^na_i$$

设该数为 $x$，首项为 $a$，末项为 $b$，且 $a>b$。因为公差是 $1$，所以可以求出项数是 $b-a+1$。

所以如果 $x$ 可以表示成：

$$2x=(a+b)\times(b-a+1)$$

那么 $x$ 就符合要求。

可以通过枚举的方式证明右边的式子中两部分奇偶性不相同，因为 $a>b$，所以 $a$ 和 $b$ 不相同。

因为右边式子必定包含奇数质因子，所以要构造出奇偶性不同的式子，就需要左边也包含。

因此如果 $x$ 没有奇数质因子，$x$ 就不符合要求，其余情况均可以用解方程的方式构造出合适的式子。

因此得出结论 $x$ 如果是 $2$ 的幂就不符合要求。

16.电信诈骗

社会考察的时候，给每个小组长下发了班级所有同学的通信表，也就是电话号。

德尔塔爆炸 同学作为我们能量最大的原子中的一个电子，开发了给别人打诈骗电话的方式，我们的内容很简单。

  • 冒充保险的客服，但是没有收费渠道。
  • 伪造出别人亲人的消息。

后来，一个叫做wy的人开发出了短信诈骗，他冒充的是楚黎教育，还是用了我的邮箱地址。

后来，我们开始和客服打电话，比如米哈游，尚海整装，联邦调查局。对于米哈游，基本是询问一通原神的问题后，用礼貌/不礼貌的方式得出结论原神降智游戏。

到了今年暑假，也就是最近。

我和其他班的几个同学又因为无聊，开发出新的诈骗。

这是我们听到几个比较搞笑的：

  • 某中年男子没有素质的谩骂，我们几个联合把他骂回去了。我们很文明，用国际社会几个臭名昭著的政治人物骂他。
