如何获取u卡  link

上一期已经完成了获取有效和更多种类的青卡的目标，接下来的目标是获取u卡。

如果你的目标是获取高质量的u卡，那么可以跳过这一章节进入下一环节了。

AH1  link

火蚁穴是获取粉卡最快的区域之一，但是可能时常有挂机车队，一般来说，在倒数第二格（u怪相对较少）是比较好的选择。

在火蚁穴的倒数第二个，最好的情况是有一个挂机点，或者一个比较安全的环境。

可使用包含以下元素的更好的抢怪：fangs，tringer，bubble。

获取的第一章ah1u卡最好可以换成coin以换取更高的输出，yucca在后期的回血装还是有一定作用的。

AH2  link

AH2的效率因为u怪的伤害比较低/有人抢u，加上安全性的因素效率可能并不比ah1差。ah2最后一格房间里面有很多m怪，在生成的uQueen后面也会召唤出很多m兵蚁。

因此ah2也是一个理想的出u点。获取的clover可以换成coin。

为了保证输出足以应对后期的需要，可以在获取u卡快速的地方获取更多的u卡。

获取海洋的粉卡  link

海洋的粉卡通常有用，而且在海洋的生物遭到了削弱以后海洋是一个比较理想的场所。

在海星区有一定数量的m怪和u怪，虽然难度较大，但是里面的u卡都非常值得获取。

  • 推荐携带花瓣：dandelion，rubber，输出花瓣，fangs。

上方的水母区也是一个很好获取u卡的地点，因为在遭到削弱之后ujellyfish数量直线下降，难度并不这么大。

  • 推荐携带花瓣：dandelion，rubber，输出花瓣，fangs，cotton，sponge。

其中粉电是一个比较重要的花瓣。

在下方的地图中，在Omaze和CK的分岔路是一个难度比较适宜的区域，主要是m/l怪，并且附件有重生点。

里面的怪比较多样，并且四周比较开阔，可以调整合适自己的难度（有全m的区域），适合作为一个长期farm的地方。

  • 推荐携带花瓣：dandelion，rubber，输出花瓣，fangs，lightning。
  • 推荐配置切换：回血电装，输出装。

沙漠粉卡  link

沙漠中最实用的粉卡是 beetle egg 和 iris。这两者都是可以用很久的，也非常重要的。

在右下角的区域通常有很多的u怪，难度较大，在沿着通道往下走，可以找一个合适的位置进行farm，但难度和效率不成正比，因此不建议在没有高dps的时候大规模刷沙漠。

  • 推荐携带花瓣：lotus，rubber，输出花瓣，fangs。

蚂蚁穴  link

AH3  link

虽然在前文中已经在蚂蚁穴获取过一定数量的u卡了，但蚂蚁穴仍然是刷u非常重要的一个地区，是从少量粉卡进阶到多数粉卡中比较重要的一环。

ah3的难度并没有非常高，但是其掉落物骨头是非常实用的前期花瓣之一。ah3挂机的人通常不会很多（没有fah多）。

  • 推荐携带花瓣：web，fangs，输出。

AH2  link

现在有了更多的海洋粉卡和高质量粉卡后，AH2的效率是可以有所提升的，用粉电获取蚂蚁后的掉落速度会更加快和多，使用更多的输出卡打m怪会更加的安全和高效率。

ubone对于m兵蚁可以打出很高的伤害，ucoin在面对低体伤m怪的时候还是可以有比较不错的表现的，因此效率可以提高。

U Ant egg 在前期挂机点时还是有一定作用的。值得获取。

AH1  link

在获取了足够的u卡片之后ah1的效率也会变得更高和更安全。

AH1的效率可以达到非常高（3hours 10kl*2+240m），因此后期也被有人看做出s的地点。

花园，下水道，丛林  link

花园&丛林  link

花园和丛林难度一般公认为比较高，粉卡也比较难以获取。

花园右下角的bee zone会刷新红和青等级的蜜蜂，效率并不高，但可以获取足够tringer。

如果想找一个难度比较低且保证效率的区域，右上方比较合适，其粉怪数量更加少。

丛林的cotton，bur，天线等是比较有用的花瓣，但是u怪比较多，难度比较大。

因为花园和丛林的粉卡难以快速获取粉卡，因此目标可以为获取足够m卡以备后时之需。

下水道  link

下水道的下方区域会有很多的u怪，难度较大，但通道前期还是有部分m怪的，蹭其他人的蛋车进去也是一个非常好的选择。

因为下水道很多怪移动速度快，因此可以携带pincer/jelly/bubble。

在进入内部之后，如果有足够的蛋车的话，可以尝试在蛋车附近抢怪。如果没有足够蛋车，可以在安全的地方放上ankh（比如一个蛋车中心点），然后尝试用pincer小心攻击其中的u怪保证安全的基础上引到蛋车附近。（目的就是不要把蛋车搞灭队了）

下一章目标便是刷蛋车和沙轮。
