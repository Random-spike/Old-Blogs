题解：P10479 匹配统计

思路  link

使用字符串哈希，然后二分答案最大匹配长度。

解题方法  link

使用了自然溢出，跑一遍 AAA 和 BBB 字符串的哈希，然后根据题意，预处理 AAA 字符串的每个前缀和 BBB 字符串的匹配长度，储存到一个 \texttt{map} 里面。（节约空间）

复杂度  link

时间复杂度:  link

O(nlogn)O(n\log n)O(nlogn)

空间复杂度:  link

O(n)O(n)O(n)

注意点：  link

洛谷评测机不接受函数名词为 hash。
