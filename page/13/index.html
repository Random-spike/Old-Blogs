在步入全青之后，接下来的目标是出粉，如果仍然有必要的稀有青没有，建议去先多刷一点。

骨头路线  link

如果你选择了骨头路线，三眼是不可少的，做好心理准备。

在tah中倒数第二个房间，是青区，跟着一些大佬挂机，效率有时候会异常的高（之前我有一次，恰好是左右两个房间都有人，刷怪效率非常高，一上午50个m骨头）但是没人的时候效率会很低，这里推荐亚服，tah人一般不多，亚服实在太多了就换eu或者na。

在你有4/5个u骨后，就去maze鬼混，混出最基本的u天线，以及必要的三眼，橘子这些多多益善。

有了这些装备，继续孜孜不倦的刷白蚁穴，刷出7骨头，就可以通过7骨 1眼 1磁铁 1回血或者装备一个速或者plank的打法 （诶嘿嘿我有5=1的废物uplank

wing路线  link

这个路线是比较推荐的，也是我的发展路线。

现在maze住着，至少9个青导弹起步。如果你觉得自己很欧，那就刷个粉导，三眼到时不必要。

如果你是青导弹afk的话，那就非常推荐在凌晨玩亚服或者找个合适的时间，找个人少的服挂机刷wing，如果人比较多，就只能在上方区域用3个mjelly挂机，一个人挂不动，需要至少2~3个人才行 但是我用2jelly1uweb1ufangs1mbulb挂机只要没有queen，我就可以通宵挂 刷出1~2个uwing，yucca可以全部烧掉换成经验。然后去打maze，刷出u导，这是非常必要的效率加强用具。再去底部和大部队挂机，这样你用导弹流也可以抢到掉落，下面的效率永远比上方高多了。

接下来，攒出50个mwing，用不太好用的垫卡公式也可以出粉wing 然后我又掉了一个 刷出4个uwing后，可以在来一根u骨防身+输出用。

接下来去maze刷出三眼以及少量u回血就够了。

导弹路线  link

导弹路线核心办法就一个：住在maze好好住着，搞出三眼以及一些必要的稀有粉后刷必要的粉。

骨头路线  link

在tah刷出大量u骨后，看到某位bingole大佬，用plank和uleaf打maze，打maze一段时间后，出了足够的回血后，去打J5出ubur，加上直接库存的ulight走上抢s路线。

u回血的作用  link

u回血前一段时间感觉特别有用，现在感觉也就那样吧。

之前刷maze刷J5都要用，现在觉得抢s路线更现实。

1.感觉不停地刷太无聊了，刷着刷着也烦了。

2.实际用处不大，经过计算，各种稀有粉出的速度都不如蹲u。

3.抢s路线效率更高

现在我打算放弃u回血，先提升dps再说

上一张说道获取更多的青卡，现在的目标则是大规模提升水平和血量，向全青和获取粉卡迈进。

获取稀有青卡  link

先列举一些稀有青举例。

刺：无疑属于稀有青，在maze刷最快，但maze难度最大。

heavy（俗称铁球，铁头泥）：在maze里面可能比刺还稀有一点，（我有30+m刺，但是heavy只有10个）这是制裁mover的实用选择

石头：跨级打很实用，dps比刺高，比heavy稍微多一点，因为掉率更高。

速，网，jelly，电：曾经不算是，现在算了，广义的稀有卡是除了wing，骨，导弹，橘子，触角，yucca，relic，agg（可能有争议，但还是算半个，毕竟这玩意只要强了可以挂机刷而且0.8%的掉落一点也不低别问我为什么掉了wing但是没掉agg但我有47个magg,18个mbasil就可以证明一个正常住在fah的人打的queen的数量，仅仅按照正常运气来说，uagg应该不算稀有，和刺之类的没法比）关于速和网的争议也比较大，但是在下水道削效率后统计，我只有25m速，29m网，比agg少多了。jelly，电就不用说了，自己算一下刷一个要多久。

额，跑题了，说道粉上去了。反正agg这东西真的很常见哦，多打打u fire queen或者u queen。

叶子，bur，cotton，root，rubber：Jungle系列，确实比较稀有。叶子你可以认为它不稀有，额，确实不稀有。

fangs，star，bubble。sand，salt，claw，pearl:海洋系列。

corn，rice，light，clover，蚂蚁穴捡漏。

powder，egg，pincer，iris，magnet：还是那句话，一起一抓一大把，白送都不要，（magnet除外）现在都是稀有的了。

剩下的比如mark，三眼，ankh这些的，emmm，过于稀有。

很多都没有列出来，比如一些回血，这个后面再说，只要知道稀有青大概是咋样就行了。

现在的目标是刷出青刺（用于刷maze），刷出至少10个导弹（多出来的全攒起来，我现在400=0u导就很烦人，7/12的197=0就太秀了）

总而言之，这一个阶段的主要目标就是打maze，橘子，触角全部烧掉（留1个足矣），大概这样疯狂的快速升级，我曾经这一步是在超高效率的下水道完成的，现在没了，我觉得打maze应该比较快（配装wing+刺+rose/star+1个dahlia）在升级到lv140后（可能需要1周的时间，但只要有毅力都可以完成）就去fah。

当然，我没说这段时间只打maze，5个青泡泡我觉得虽然有难度但是是必须的，在螃蟹区蹲u泡泡或者在上方打红泡泡，用垫卡的办法也可以出一点。light，rose在你泡泡足够是可以在maze或者c3蹲ladybug蹲出来的。其他的，可以移步上一张或者私信骚扰。

努力升级提高综合实力  link

在maze打的头晕目眩的时候，可以自我放松，在florr中到处逛一逛，最好收获mstarfish，如果任然面临着稀有青出青困难的问题，可以打u怪掉落。

那在这里鬼扯一些对未来的规划和基本的认识。

在全粉（除去三眼，世界树，磁铁）后，前期3个路线可以走。

1.wing路线

2.骨路线

3.歪门邪道的导弹路线

wing路线靠住在fah的基础上，加上回血完成，三眼成为不必要的，虽然有了更好，但不会因为没有wing就是废物。主要难点是fah抢怪环境困难，建议在大挂机点上方开辟青卡挂机池，效率不高但是有掉落。

骨路线是曾经参考bingole的，三眼成为必要的，好像还带了个plank，青的也可以，啊哈哈哈哈我有粉的，效果也不错，但伤害和综合实力没有wing高，但有点在于，fah抢怪简单啊。

导弹路线，这个嘛就是我们班的一个小菊花同学，200不到出了5个，这让我很无奈，虽然u导是fah抢怪必备，但我死活不出也没办法，Virgo曾经好像也是这样发家致富的？这个配置单刷能力可能差一点但用机关导抢s怪可以说是dps最高的廉价配装（高dps的light，sand流也太贵了）

题目大意：  link

题目给出一个数列，求数 ddd 使得这个数列每一个数除以 ddd 的余数都相等。

思路分析：  link

数列的的第 iii 个数除以 ddd 的表达式为：

q×d+r=niq\times d+r=n_i q×d+r=n​i​​

因为题目要求每个表达式中的 rrr 都相等，所以每两个数，
例如 iii，jjj 之间作差可以得到：

(qi−qj)×d=ni−nj(q_i-q_j)\times d=n_i-n_j (q​i​​−q​j​​)×d=n​i​​−n​j​​

用同样的办法操作数 iii，kkk 得到以下式子：

(qi−qk)×d=ni−nk(q_i-q_k)\times d=n_i-n_k (q​i​​−q​k​​)×d=n​i​​−n​k​​

题目要求 ddd 的最大值，因为 qqq 的值都是可以跟随 ddd 值变化的因变量，而 nnn 的值是题目给定的常量。所以 ddd 的最大值就是：

dmax=gcd(ni−nj,ni−nk)d_{\max}=\gcd(n_i-n_j,n_i-n_k) d​max​​=gcd(n​i​​−n​j​​,n​i​​−n​k​​)

重复执行如此操作，设数列长度为 mmm，相当于可以转换为以下表达式：

d_{\max}=\text{gcd}(\vert n_1-n_2\vert,\vert n_2-n_3\vert,……,\vert n_m-n_{m-1}\vert)

系列文章

现在你有了一些青卡，现在的目标就是获得更多种类的青卡和更多的青卡。

获得更多的青卡  link

要想完成这一步，就需要有磁铁，最好是红磁铁，因为海洋比较远，所以最好一次走完（

但接下来的florr会有趣的多，以下步骤可以不分顺序，但都要完成。

第零步：疯狂烧卡，把所有幸运草全部烧掉，包括青的，如果还是不够，烧掉所有的红peas，以及所有紫卡。

目标：craft等级，可以合成青卡，槽位等级，10槽，血量大于7k。

海洋路线  link

第一步：去水母区，但去水母区之前要在路上拿几个磁铁，哪怕是蓝磁铁，因为这样可以让你的磁铁导青水母的电，让你减伤。

第二步：路上再捡几个海绵，稀有级别为红。

第三步：到达水母区，刷青电，青jelly（后面有大用处），刷红磁铁。

推荐配装：3magnet(记得打的时候不要让磁铁本体打在水母上，1sponge（减去90%的伤害慢慢的放出来），尽可能多的mwing，尽可能多的mleaf，最好一个红速

第四步：用m电+mleaf去水蛭区刷mfangsmfaster，水蛭区是红区，所以基本不会有问题。尽可能的用mfangs替代mleaf。带1~2个mfaster。

第五步：去亚服（人最多）海星区刷海星，这可能比较需要技术，如果实在不行，那就返回水蛭区，向水蛭区的下面jungle走。如果可以，那么刷2青海星走人。

海洋路线到此结束，预计时间，2~4小时

丛林路线  link

为什么是丛林呢？因为海洋离丛林最近（

丛林里面在这个阶段可以刷的不多，但在亚服J5跟着大部队收集mleaf，mcotton，m天线，miris。

丛林路线结束，预计时间1小时。

蚂蚁穴路线  link

现在，从丛林的入口进入白蚁穴，出身点应该是红紫区，逐渐向上，可以看见青区，但是成群结队的白蚁非常难以击败，你很可能会死很多次，所以快速刷出1-2个青骨头就撤，骨头用处其实也没那么打，要跨级打才强。至于粉骨，三眼就是必要的了，他吗这玩意真难刷，我5000m蜘蛛，40u蜘蛛都不掉，但wing的限制就不大了，后期主要是打J5和打maze，uwing对付最烦的怪：黄蜂，效果自然要比骨头好一点，但u骨迅速消灭时刻想要如车轮碾死你的ladygaga和蜘蛛。另外说一句，挂机u骨达到极限效率的效率是惊人的，（我当时只有2橘子2uwing1ufangs做dps，说明挂机点抢怪速度不快），我在一个上午的时间内，挂机刷出了50青骨，然后…

这是后话了。

走出白蚁穴（走左上路线，到达熟悉的ah2后走左下的传送门），往fah右下走，可以看到红，青红火蚂蚁，可以使用，电，骨头，cotton，jelly来保证自己的安全，fangs用来回血。

在火蚂蚁穴，刷出更多的wing，直到大约6个，yucca全部烧掉。

花园路线  link

接下来是花园路线，蹲uladybug，如果你抢不到，那就去海星区刷海星，或者丛林刷dahlia，有了好的青回血（leaf，yucca，shell不算在内）就可以去花园红区，或者maze，目标很简单，刷出9个导弹（青的）为以后做准备

路线攻略结束，为什么不放沙漠的呢？因为我觉得沙漠实在废物，前期效率比较低，比较容易死。我也懒得写了到此结束（）

发展建议1请走 这里

现在，你已经获得了大量的ant hell2的红卡，接下来的目标是获得青卡

第一张青卡  link

如果你是欧皇，打u怪爆出来一张，那请你也不要跳过这一章（

在ant hell2 逐渐向右推进，你可以逐渐看到更多的青怪，用你的红石头，红玉米，红叶子，造成比wing更高的伤害，用粉骨头，粉wing的人可以直接秒掉青怪，所以尽快造成5%的伤害，大概12k，这，有点难抢罢 …

如果你死了，你可以C5蹲到ultra centipede，这个比较好抢，不停的拿agg怼着它就好了。

加技能点：点开合成青卡，继续升级槽位。

最终，在ant hell兢兢业业的刷，很快就可以出青

到达出青自由  link

到达出青自由的含义是：出青不考运气5=1，或者靠爆，以及各种奇怪垫卡（

这个比出一张青卡的难点是需要提升效率（doge

比如说赶路吧，一张图一张图的赶显然非常浪费时间，就可以考虑去O7刷点lag泡泡，如果你之后想要m泡泡，建议去美服水母区（没人）或者蹲u泡泡效率高。

为了以后升级到10槽做打算，有5-10个红泡泡最佳。

有了红泡泡，就需要有mlight或者黄agg（主要是reload时间比较长）来压住泡泡，可以弹得更远。

mlight刷法：蹲uladybug（c3）6%会掉，但死的实在太快了，不好抢，那就只能从蚂蚁穴的babyant掉落，合成了，maze在全青以前，刷light的效率还是很低的。

有了泡泡，light，那么你赶路的速度就快了很多，可以更频繁的出入蚂蚁穴，这样就可以出更多的青。也就是说，为了出青的效率，最好牺牲游戏的乐趣，住在蚂蚁穴（以后出粉也将会是这样）

如果你不愿意牺牲游戏的乐趣，那么可以去C6（可能编号有问题，它是刷uhornet那个区，不知道的看发展建议1 的地图。）c6是花园红区，也有少量的青怪，里面可以刷，蚂蚁穴，黄蜂，蜜蜂，石头，毛毛虫，蜘蛛。这也可以让你出青，但是这里面的青卡大部分用处不如蚂蚁穴的大，额当然青刺以外，以及青天线是很必要的。另：在这里遇到uhornet最好直接泡泡逃跑，不然白白死了还要重跑。

从头开始出第一个红  link

刚开始的一个号，最好先打出生点c1，全身黄装之后去打左边那个不知道叫啥的区域（

在旁边c2打出最基础的一些装备，至少要有回血和输出罢 后去打下水道，最好跟着一个egger走，小心一点打，打出第一个紫wing，死了重跑

有了紫wing，在c2继续往左走，达到至少全蓝，去左边的c5，走那个底部传送门，去ant hell

地图上标出了，到达ant hell 后，上面那个区域是紫区，但实际上是蓝区，蓝区的怪应该基本可以打的过，然后把中间的蚂蚁蛋打掉，蓝以下的不要管，如果出了紫的，拿出来用，蚂蚁蛋是接下来发展的关键。

继续往右边走，紫区，有很多蚂蚁蛋，不停地打紫兵蚁，打出全紫wing，clover全部烧掉换经验，在这个小区域等待蚂蚁蛋的不断的刷新，刷出第一个红蚂蚁蛋。（不管你出了啥红，那还是需要一个红蚂蚁蛋）

到达红卡量产  link

在有了第一个红ant egg（简称agg）后，使用它和wing。你可以继续向上，上方会刷红/紫 怪，相比于兵蚁，你更需要打工蚁和baby ant，他们的掉落对你更有帮助。

这时候，插播一点技能点方向点法：合卡你最好不用点（前期），先点槽位和最大血量。为了以防zms过来找碴，我解释一下,前期的合卡对整体发展帮助不大，红卡不应该全是掉落的吗

在这里，刷出红叶子作为回血，红玉米留着（会坑死自己的），其他卡都留着，出了clover烧掉，不断的往左走，打红等级的怪，青怪先别打，抢不到（

当然，死了很正常，你可以在c1蹲ultra rock的刷新，凭着全红的装备，如果你找得到的快，应该可以有掉落，红石头和红重铁头 都很有用
