省流量：

今天要说的重点，就是：

想从头学信息学奥赛，奥数生好转么？

这个问题，还是比较好回答的。

但在说之前，还是要铺垫一下，看一下现在信奥是个什么情况。

拿北京举例：

今年的信息学奥赛中，北京初中联赛的参赛人数，已经达到了7600人。

听起来挺多，实际上也挺多的。

这是个信息奥赛开展39年来，北京参赛人数历史最高，没有之一。

尽管如此，这个人数，和奥数动辄好几万人参加的盛况都是没法比的。

于此同时，高中联赛的参赛者，只有2900人不到。

您可能觉得，毕竟是高中联赛，参加的人少也是正常的嘛。

错了。

一般来说，参加初中联赛的比较有实力的奥赛生，也会顺手报一个高中联赛。

这种被称为双开的操作，很普遍。

虽然我们把CSP-S叫做高中联赛，但这是从难度上和主要面向对象来看的。实际操作中，上至八旬老人，下至幼儿园小朋友，只要您愿意出这50块钱，就能报。

就看您需不需要了。

对于八旬老人来说，肯定是不需要，因为不需要升学，也不需要找工作。

对于幼儿园小朋友，也一定不需要，因为学不会。

这种中间，CSP-S报名的主力，高中生外，主要是初中生。

因为只要是初中生，基本上都是两年以上的奥赛生，提高组的知识多多少少肯定都是会一些的。

所以，通过高中联赛的人数，就能看出来这个区域的正规奥赛生（两年以上）的人数规模。

高中联赛的人数，基本是正规奥赛生人数的150%左右。

正规奥赛生包括但不限于：

1- 重点中学的校队

2- 信息学奥赛机构的学生

3- 或者，只要学了2年以上C++的都算

刚刚说了150%，这多出来的50%，大多是新晋奥赛生第一次参加比赛，为了获取考试经验而报的两个比赛（J和S）的新手。

所以，通过简单计算：2900 / 1.5，这就得到正规奥赛生的人数，是1933。

这里面低年级的正规奥赛生，不管是小学的还是初中的，最少能有700。

剩下的1200人，应该都是高中奥赛生，分布在几十个区重点（外省的市重点），市重点 （外省的省重点）学校中，平均一个学校几十个学生。

数据完全说得通。

也就是说：

1- 高中联赛的高中生，1200左右

2- 高中联赛的初中小学生，700左右。

那么问题来了，如果双开高中联赛的正规奥赛生就700，满打满算，只考初中联赛的正规奥赛生，也就是3倍，2100人…

这一共才2800人。

那，报名初中联赛的7600人里：

剩下的4800学生都是哪来的？

保守估计，这里面得有2500都是小学生。

肯定是从奥数移民过来的。

要说这一波小学生，也是蛮惨的：

1- 想要考奥数，大师赛没了

2- 想要考信奥，分数线太高

3- 想要学信奥，不知道自己能不能行。

前两条，是既成事实，我们改变不了，不过这第三条，我们可以分析一下。

那就是今天的重点了：

想学信奥，以前的奥数生好转么？

我的答案是肯定的：

好转。

没有转折，没有所谓的但是，就是单纯的好转。

我下面来具体举例说明为什么：

这里不妨看一下五年级的奥数样题：

鸡兔同笼问题

1．鸡与兔共100只，鸡的腿数比兔的腿数少28条，问鸡与兔各有几只？

这道题可太经典了，可谓是千年传承。

1500年前的《孙子算经》里面就记载：

【今有雉兔同笼，上有三十五头，下有九十四足，问雉兔各几何】

这段文言文，还是比较容易懂，意思是说，有兽与禽同笼，上面看有35个头，下面看，有94只脚，问兽、禽，各有多少只。

这个正确做法，就是用函数。

x是雉，y是兔。

那么x + y = 35，因为上有35头。

同时，2x + 4y = 94，因为下有94足。

化简得到：x + 2y = 47

也就是：x = 47 - 2y

那么：47 - 2y + y = 35

最后：y = 47 - 35 = 12

x = 47 - 24 = 23

这是正解。

小学生的理解难度，我认为是5，如果20分满分。

（本篇的评分，都是针对小学生的，不针对成人）

随后，我们可以看看网传的奥数做法。

用鸡兔同笼的网络版数据：1．鸡与兔共100只，鸡的腿数比兔的腿数少28条，问鸡与兔各有几只？

网络解法是这样的：

解：

4*100＝400，400-0＝400 假设都是兔子，一共有400只兔子的脚，那么鸡的脚为0只，鸡的脚比兔子的脚少400只。

400-28＝372 实际鸡的脚数比兔子的脚数只少28只，相差372只，这是为什么？

4+2＝6 这是因为只要将一只兔子换成一只鸡，兔子的总脚数就会减少4只（从400只变为396只），鸡的总脚数就会增加2只（从0只到2只），它们的相差数就会少4+2＝6只（也就是原来的相差数是400-0＝400，现在的相差数为396-2＝394，相差数少了400-394＝6）

372÷6＝62 表示鸡的只数，也就是说因为假设中的100只兔子中有62只改为了鸡，所以脚的相差数从400改为28，一共改了372只

100-62＝38表示兔的只数

我换成斜体，是因为这个解法简直太恶心了，有必要和我的正文区分开。

用函数秒懂的事情，一定要用混乱的言语来教学。

这就是奥数一贯的问题：偏重巧算，忽视原理。只教巧算，是有代价的，那就是，需要完全避开早已经成熟的数学原理。

尤其是这题。为了把这个372 / 6的例子说通，就非要用一个很拧的逻辑，那就是把兔子换成鸡。

这种解法，对比函数来说，一定是差的。之所以看着特别不舒服，就是因为，要理解这个解法，那无限接近于听一个智力有障碍的人说话。一个智障给你解一道题，还要求你确认明白他的做法，这不是折磨是什么？

而且说到底，这种做法也只能解决鸡兔同笼的变种：脚的差值。

那如果不给你差值呢？就原题，35个头，94只脚，你还会做么？

不会了吧。

理解难度，就这道题的题解文字上来说，对小朋友而言，我认为能飙升到18（20分满分）。

这里要注意，不是分越高越好，而是分越低越好。越低，就好理解。

越高，就越难理解，就越差。

这很有可能是奥数教学中的一个误区，认为小学生，不能理解复杂的概念，所以一定要用看似浅显，实则冗长无比的废话来教学。这实际上增加了学习难度。

这也有可能是网传的解法太差了。

至于是哪个，都不重要。

至少，从我信奥教学的经验来说，如果学生是五年级的孩子，帮助他们理解函数正确概念，这很容易，一点都不难。

那么，我们现在看看信息学奥赛的解法：

1- 一共100只，那么，我们定义两个变量，一个x一个y（这第一步，和数学的正解一样）

2- 随后，根据题意，逐一枚举x和y的数值：也就是说，如果x为1，那么y一定是99，同理，如果x是2，y一定是98。我们很快就能发现，实际上真正变化的，就只有x而已。

于是：

3- 写一个循环，让x从0循环到100 （0只鸡，枚举到100只鸡），逐一计算（4y - 2x），算出来腿数差28那就立刻停止。

是不是简单？

我觉得不会编程的家长，看着都觉得简单。

理解难度，我认为也就是2。

这里面唯一需要做得，就是把C++的计算机程序写出来而已。

这样的奥数和信奥的类比例子，还有很多。

在初中联赛这个阶段，结论都差不多，都是信奥比奥数简单。

然而，简单归简单，需要知道的事情是，鸡兔同笼这种问题，在初中联赛复赛中，如果做对了，怎么都能拿个三等奖。

北京小学生，能拿到初中联赛三等奖的，也就是几十个学生。

这个数量级，可以说相当少了，堪比奥数的一等奖。

所谓物以稀为贵，就是这个道理。

所以，要记住两件事：

1- 信奥难度 < 数学难度 < 奥数强加的理解难度

2- 信奥奖项含金量 > 奥数奖项

如果奥数生能记住上面斜体字那些幺蛾子的解法，还能做出题来，那，掌握信息学奥赛的解法，显然可以轻而易举。

所以，奥数，信奥，您说学哪个？
