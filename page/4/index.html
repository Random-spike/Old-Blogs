题解：UVA406 Prime Cuts 题解
2024-08-10 - /032_题解：UVA406 Prime Cuts/

首先看一下题目一眼平衡树。这里写了 Treap\text{Treap}Treap。

题目思路  link

根据题意，要求写出在前 nnn 个数中在要求范围内的质数，因为有多组询问，所以考虑预处理所有在范围内的数，因为 n<1001n<1001n<1001 所有只需要维护前 100010001000 个数，因此写一个稍微暴力一点的质数判定也是可以接受的。

接下来还有要处理的问题是：

  • 平衡树如何使用。
  • 平衡树如何给出答案。

解决方法是在预处理的时候将所有质数插入平衡树中，在输入 nnn 的时候可以直接给出所在质数序列的排名。

对于第二个问题，可以用二叉搜索树的性质计算第 kkk 名的答案，当然也可以用后继的方法查找（理论上应该更快一点？），但是这里数据范围比较小可以直接用比较简单的写法。

最后要注意的点是要保证输出的数有意义，也就是说不能输出很多重复的 000 或者大于限制的数。

Ⅱ florr攻略与指南 florr

下水道 丛林 火蚁穴  link

上一期文章中介绍了在完成全青后的花园，沙漠，海洋，黑白蚂蚁穴的打发，这期接着介绍其他区域。

下水道  link

下水道的怪物种类比较单一，攻击方式也比较统一，掉落物相对不那么实用。

  • Fly。攻击方式为碰撞，以扭曲形式的步伐向玩家进攻。克制方法：bone/pincer/egg/jelly。掉落中wing在前期比较实用，poo一般在跑路装使用。

  • Roach，原名cockroach。中立生物，移动速度非常快，在多数情况下不建议直接正面硬钢。克制方法：pincer/egg/jelly。

  • Moth，友好型生物，不会对玩家攻击，一旦被攻击后会和fly类似的运动方式逃跑，掉落物实用型一般。

  • Spider，敌对型生物，leg及以上的蜘蛛会在路径上放一张同等级的网，克制方法：bone/jelly/egg。

下水道主要值得获取的卡片是 wing。wing在打低等级/体伤的怪时候常常有很好的表现，虽然wing的伤害不是非常高，但也是比较实用的花瓣之一。

丛林  link

算是比较难打的地图之一，主要原因可能是因为玩家稀少。里面有比较多的实用的掉落物值得获取。

丛林的生物移动速度除了firefly都没有玩家快。

  • leafbug。技能是有护甲（和bone类似），一般情况拿输出装可以解决。掉落物root可以给花朵本体增加armor。

  • bush。和花园的rock，海洋的sponge类似，掉落物cotton配合sponge有很好的效果。

  • wasp。丛林主要有威胁的生物之一，经过更新之后导弹的方向非常诡异，在距离wasp一定的数值的时候导弹是打不中你的，可以用wing比较安全的解决。

  • mantis。丛林主要有威胁的生物之一，发射出的绿色小球是给花朵本体减少armor，换句话说，被debuff后，花朵受到每次的伤害都会变多。可以使用wing/fangs/高血量高输出花瓣快速解决。掉落物bur在之后会有比较大的用处。

  • Firefly。在本体受伤害的时候回放电。

  • dark ladybug。和花园的类似，掉落dahlia有用。

  • mound 丛林的蚂蚁穴，难度并不大，在危险时候可以进去躲避。掉落物rubber有用。

丛林的主要威胁在于被mantis和wasp夹击。因为丛林的生物的射频都非常高，很难直接正面反击，在mantis的debuff下伤害会非常高。这时候比较好的解决方式是用远程花瓣wing/组队配合。

有一个技巧是使用relic时候孵出的是白蚁蛋，可以尝试孵出白蚁蛋（和召唤物共享伤害），本体携带battery，cotton，salt有必要携带cactus/soil，sponge。实现反伤流打发。（来源于vowov&zhhhh Firefy）

火蚁穴  link

火蚁穴的出卡效率非常高，难度也比较简单，但卡的质量并不高。

在火蚁穴底部用挂机点，可能在中间区域也有，进入时可以跟随前来挂机的人比较简单的实现，如果是只有一个人时，使用web，pincer，bubble和比较好的技巧也是可以没有难度的进入的。

因为火蚁穴有很多青怪和蛋挂机池。所以可以带足够的搞高伤害花瓣在附近抢掉落。为了更高的效率可以将reach点满，血量尽可能多。

可以使用以下几种花瓣：stringer，iris，claw（必须在怪80%血量以上攻击），rock，fangs（回血），bone这些花瓣通常比较常见。

在这里可以获取足够数量的yucca花瓣和sponge花瓣配合使用。

更快速的换装  link

在使用了sponge之后，最好需要一个专门的回血装。泡泡装，输出装也需要储存以方便使用。

游戏内置了快速换装的按键。在主槽和副槽是r键，单个切换时数字键，而整个切换的是l+num。

具体来说就是可以预先在背包配好一套装备，在同时按下L+shift+num 后可以储存（num为键盘上0~9的一个数字），在调用的时候按下 L+对应的num即可。

比如输出装把sponge放在槽位的第10个位置，当掉血速度太快时，可以快捷操作回血装（回血装的第十个位置必须是sponge）进行回血操作。

特殊的技能点  link

观察技能树，发现在一些普通的子树上会有一些奇怪的分支。

这里介绍一下常用的内容。

在槽位子树的末端是磁铁技能，这个技能可以让你不携带磁铁获取更大范围的花瓣掉落，这个技能在farm的时候比较实用，抢s用途并不大。

在槽位子树上还有花瓣复制的技能，这个技能可以让本身有多个部分的花瓣再多出一个部分，这个技能一般在抢s的时候配合沙轮用，或者对有sagg的人用，比如3个sagg拥有者SES-lightning（已退游）。

在血量子树上有second chance技能，这个技能非常实用，不论是在farm还是抢s。可以让你在受到致命伤害的时候不会死亡，保持一秒的无敌后保持一个很低的健康值。（m等级的可以再无敌时间大幅加快回血速度）

毒伤和体伤的特殊技能基本是对该技能的强化，前期（除非撞击流）比较少使用。

获取高端青卡  link

在对地图的每个区域有一定的了解之后应该对花瓣有更多了解。

不同花瓣都有不同的用途，一个良好的配装中应该有不同的功能。

在前期比较重要的两个方面一个是回血，一个是输出。

海洋  link

回血花瓣最好用的便是starfish和fangs（应对海洋沙漠蚂蚁穴）。为了获取更多的starfish，海星区便是效率最高的地方。

海星区在地图右上角，到达海星区要途经水母区。（曾经海星区和水母全是u怪，效率非常高，后来一段时间是少部分u怪带很多m怪，现在已经m怪都不多了）海星的技能回血非常烦人，可以使用dandelion阻止它回血，因为海星血量低体伤低，移动速度也没有玩家快，因此基本构不成威胁，在可能被包围时可以直接泡泡弹出去。（其实到了后期被m怪包围也不会死）在海洋的回血使用fangs是最佳选择。

当然在途径的水母区也值得一去，闪电的花瓣可以穿甲攻击，也可以对leech造成非常大的伤害。

海星区可以获得沙子，海星，泡泡，电，等非常多的实用花瓣。

花园  link

花园中rock，stringer，rose，light，leaf等很多花瓣都有用，非常值得一刷，主要问题是难度比较大。

高等级黄蜂会有预判的操作，因此可以快速摇动鼠标的左右方向一步一步向前推进进攻。面对黄蜂的远程攻击最好技能点点满reach。

蜘蛛非常克制reach的技能，可以使用骨头快速击杀，在无法快速击杀的情况下可以使用jelly/组egg队。

蜜蜂的轨迹和fly相似，因为其血量低，可以通过边后退边输出的方式消灭蜜蜂。

石头，蜈蚣，蚂蚁穴等怪通常不构成主要威胁，可以使用电快速获取更多掉落。

蚂蚁穴  link

蚂蚁穴中比较有用的花瓣是骨头，clover和glass通常会被选择来交易换成coin。

Glass在面对大群投射物的时候有比较好的格挡效果。

丛林  link

虽然丛林的难度比较大，但是其中很多花瓣都是非常有用的。

上文已经讲了如何面对丛林的怪，这里详细讲讲不同花瓣的用途。

  • Missile 远程攻击。

  • ante 通常在跑路装使用，可以保证跑路的安全。

  • blood stringer 在攻击的时候减少自己的血量，和fangs配合使用比较好。

  • cotton 用于吸收毒伤或者海绵伤害。

  • leaf 回血+输出。

  • root 单人挂机的时候个更安全。

  • bur 配合光轮/沙轮使用

  • peas 没啥用。

  • dahlia 对频繁远程攻击的怪使用。

  • yinyang 泡泡。

  • battery 对水蛭可以使用/碰撞 可以使用。

在本章节结束时，希望你已经获取了大部分种类的青卡并且做好获取u的准备了。

Ⅰ florr攻略与指南 florr
2024-08-07 - /034_Ⅰ florr攻略与指南/

前言

florr.io 在2024/4/13 进行了较大规模的更新，在更新之后产生了百年未有之大变局。

鉴于目前没有系统的攻略，指南为玩家提供发展路线的指引，公会的选择，因此虽本人能力有限，但在老版florr攻略的影响下决定进行重铸。

因为曾有比较长的一段时间脱离了florr，因此可能产生疏漏，若评论区指出，将感激不尽。

如何加入更多的florr社区？  link

加入florr社区是florr玩家的比较重要的一个环节，毕竟在遇到困难/获取信息方面florr社区常常有至关重要的作用。

  • 中华好花花

    作为洛谷本地的较大规模florr玩家社区，是长条地图，wave制florr玩家bijiachen1222建立的讨论平台。内含有包括大量相关资源。

  • Q群 florr official 765884346

    该群是florr官方讨论平台之一，对中国大陆用户来说使用方便，QQ也是很好的平台之一，也可以选择可以加入各大florr相关群聊。

  • discord

    是florr官方讨论平台之一，对中国大陆用户并不那么方便，但是内涵很多高质量频道资源。

  • Wiki

    florr wiki 可以提供很多关于数据的信息，但比较严重的问题在于信息有疏漏或者过时。

这里提供世界地图以供查阅。

获得第一张红卡  link

第一次进入在florr的花园出生点会在出生。死后也将在这里复活。在出生点可以很快速的获取 包括 common，unusual稀有度的卡片，如果继续往出口方向走，获取rare稀有度的花瓣并不困难。

如果觉得黄蜂很难打，那么你可以试试看orange的挡导弹效果，stringer的爆发高伤害效果对它们的克制。

在有了基本的卡片之后，可以借用周围玩家的力量/系统匹配的squad 更快速的获得卡片。

在有一定的基础过后，可以向花园蚂蚁穴的入口推进（地图上标明了），进入蚂蚁穴后面对的是较低伤害的怪和相对丰富的掉落。有需要的有glass，antegg，leaf，corn，light，rice等花瓣，在低难度的区域获取红卡。

提升等级  link

在lvl10的时候可以拥有更改用户名的机会。

注意：这个机会是唯一一次，修改名字一定要慎重。

可以点开左上角的图鉴查看不同怪物获得的经验值。鼠标移动到你花血量下面的可以查看具体的经验值。

完成成就  link

在左上角红色的按钮点进去是成就部分。

完成一个成就会给你提供一个技能点（TP），关于如何完成可以看下方英文描述。

一般来说，成就完成难度是根据稀有度排列的。

短期技能点  link

技能点目前是可以无成本随时重置的，也就是说，你可以在不同区域调整技能点点的方式。

一般来说，在前期，槽位（loadout），reach，血量是相对重要的，其次重要的是reload。

比如说在蚁穴，可以点更多的血量，reach保证安全的情况下点尽可能多的loadout。

不同的区域之后会在可及后一一解释。

下水道  link

接下来观察地图可以发现，以目前的装备（少量红卡）来说，到达下方的沙漠，右边的海洋是比较有挑战性的事情。

在除去蚁穴之外，另一个不同的地图是下水道，下水道中主要有用的花瓣有wing faster web，（触角虽然有用但相比来说比较稀少），可以再获取足够的leg wing等花瓣之后继续刷蚂蚁穴。

白蚁穴  link

再有了一定数量的红卡之后可以通过刷蚂蚁穴的方式获取骨头花瓣。

骨头花瓣有护甲，护甲的作用让骨头花瓣收到的伤害为（\max(0,\text{目标怪体伤}-{护甲值})）。

换句通俗的话说，骨头可以在本身减少少量血量的情况下对低级怪产生非常大的伤害，比如说，青骨头会对红蚂蚁产生等级压制。

获取青卡  link

纵观所有地图，最好获取青卡的地图是黑蚂蚁穴，在右上角倒数第二格的区域有非常多的红怪。对于这一步的玩家打起来是比较安全高效的，比较理想的方式是在造成一定的伤害后引怪给挂机的人。

在附近地图还有一种比较有用的花瓣是 ant egg，低级该花瓣召唤出的 4 个蚂蚁可以起到比较好的辅助效果。

在黑蚁穴获取青卡之后（有clover，glass，leaf等等）可以尝试去沙漠，海洋地图。

获取更高级的青卡  link

在黑蚂蚁穴获取的青卡通常用处并不多，在白蚁穴获取的骨头会更加有效。

如果你是一名热衷于探索的玩家，可以到达海洋地图，沙漠地图获取其他种类的卡片。海洋地图可能相对简单，在左边分岔路向上走面对的怪更多是水母，而向下走面对的是更多样的怪物，比如说泡泡，螃蟹，海星，海绵，水蛭。

海洋的青卡相对于沙漠的可能更加有用。在分岔路向上走可能需要更多的回血。向下走需要的更多是技术。

这里介绍一些怪的打法。

  • 泡泡，威胁很小的怪物。掉落物泡泡在跑图赶路的时候非常有用。

  • 螃蟹，可能比较有威胁的怪物。掉落物claw，sand都比较有用，可以点更多的reach等级和输出解决。

  • 水母，可以使用rubber（来自丛林蚁穴）抵消电的伤害，但如果没有rubber可以用高血量伤害的卡片+fangs或者回血保守打法。

  • 海绵，血量较高，建议在安全环境下解决。

  • 水蛭，掉落物fangs实用，在被吸住的时候很烦，可以使用闪电快速解决。

  • 海星，掉落物sand，starfish很实用，使用dandelion（来自花园）限制它回血。

  • 贝壳，掉落物磁铁实用，使用输出解决。

沙漠  link

沙漠中比较有用的掉落物是毒（iris），beetle egg，pincer，cactus，ygg，粉末。

这些卡片通常没必要大量获得，所以前期个人认为海洋重要性更强。

这里介绍一下打法：

  • 蝎子：可以使用莲花，fangs，cotton解决。

  • 甲虫：使用pincer可以很简单的解决。

  • 仙人掌：保证安全的情况下解决。

  • 沙漠蜈蚣：一般靠爆发输出解决低等级。

注意：以上怪物打法一般是低等级用的，也许并不适用于高等级。

赶路装  link

在探索的范围逐渐变大后，赶路成为重要的组成部分。

通常用法是阴阳泡泡。

包含以下因素：触角，阴阳，多数泡泡，尽量高等级的light。

通过连续按数字按键切换阴阳保证泡泡尽量在同一位置，鼠标右键激活泡泡。

如果前期操作困难可以降低花瓣转速（Q键）
