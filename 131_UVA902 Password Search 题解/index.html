2023-08-18 - /131_UVA902 Password Search 题解/

题目大意：

多组数据，每个数据给定一个字符串 $s$，求出在每个长度为 $n$ 的子串，哪个子串出现最多。

解题思路：

遍历字符串 $s$，使用 map 容器储存每一个子串的出现次数，同时进行比较取出最大值。
