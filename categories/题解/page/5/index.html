P4136 传送门

思路分析：

数学染色问题。

在这个棋盘上以左上角为原点建立平面直角坐标系，棋盘上第 $i$ 点的横坐标是 $x_i$，纵坐标是 $y_i$。如果 $x_i\times y_i$ 是偶数，那么染黑色，其他棋盘上的格子均为白色。小明走到棋子只可能落在黑格，小红的棋子只可能落在白格。

小明要获得胜利，就必须要求右下角的格子染的颜色是黑色，所以相当于要求 $n$ 为偶数，反之亦然。

How Many Knights 题解 题解
2023-07-21 - /136_How Many Knights 题解/

原题 How Many Knights 传送门

题目大意：

骑士（马）在象棋中走“日”字攻击，可以攻击到不在同一行或者同一列且曼哈顿距离为 3 的格子。

现在给定一个 $M\times N$ 的平面，求出最多放多少个骑士在这个平面里面。

思路分析：

这是一个规律题，经过推算和尝试后不难发现以下结论。

设置变量 $ans$ 为可以放马的最多数量。

$$
ans = \begin{cases}
M &\text{if } N=1 \
N &\text{if } M=1 \
\lfloor\frac{N}{4}\rfloor\times4+2\times\min(N\bmod4,2) &\text{if } M=2 \
\lfloor\frac{M}{4}\rfloor\times4+2\times\min(M\bmod4,2) &\text{if } N=2
\end{cases}
$$

其余情况下：

$$
ans=\lfloor\frac{N\times M+1}{2}\rfloor
$$

CF1462D Add to Neighbour and Remove 题解

原题 CF1462D 链接

题目大意：

输入 $T$ 组数据，每组数列 $n$ 个整数，然后合并 $a_i+a_{i+1}$ 至 $a_i$ 位置。最后，使得数列 $a$ 所有元素相同。

思路分析：

这道题目数据不大，$1\leq n\leq3\times 10^3，\sum n\leq3\times10^3$，意味着可以接受时间复杂度为 $O(n^2)$ 的暴力枚举。

枚举过程：求和 $a$ 数组的和为 $sum$，从 $k=n$ 开始从大往小枚举，易知 $sum\bmod k=0$ 同时创建函数判断 $k$ 值是否成立，成立输出 $k$，否则 $k=k-1$ 继续枚举。

P9412 「NnOI R1-T1」购物

原题 P9412 「NnOI R1-T1」购物 传送门

题目分析：

题目中给出欧艾国硬币面值数量 $n$ 和面值。（注：它们的面值为 $1=a_1<a_2<a_3<\cdots<a_i$ 且满足 $a_{i+1}$ 是 $a_i$ 的倍数）

求出最小的 $m$，使得买 $m$ 元的商品需要的硬币数比买 $m-1$ 元的商品需要的硬币数更少。

思路分析：

  1. 当 $n=1$ 时，只有面值为 $1$ 的硬币，显然无解，输出 $-1$。

  2. 当 $n=2$ 时，如果 $a_2$ 的面值为 $2$，那么购买商品需要 $\lceil\frac{m}{2}\rceil$ 个硬币，无解，输出 $-1$。

所以当 $a={1}$ 或者 $a={1,2}$ 时，无解。

同时得出 $a_2\ge2$。

  1. 当 $a_2=2$ 时，且 $n\ge3$，那么 $m=a_3$。

  2. 当 $a_2>2$ 时，$m=a_2$。

CF1823B Sort with Step 题解

CF1823B Sort with Step

原题 CF1823B 传送门

题目分析：

题目中给定 $T$ 组数据

在每组数据中：

  • 第一行是数列的长度 $n$，以及模数 $k$。

  • 第二行是乱序的一个公差为 $1$ 的等差数列。

然后做两个操作：

  1. 交换一组数： $p_i$ 和 $p_j$，其中 $\lvert{i-j}\rvert\bmod k =0$。

  2. 如果以上步骤不能让 $p$ 数列有序，那么可以交换一次 $p_i$ 和 $p_j$。（以下简称其为第二操作）

输出数据：

  1. 如果不需要第二操作就可以让数列 $p$ 有序，则输出 $0$。

  2. 需要第二操作后可以让数列 $p$ 有序，则输出 $1$。

  3. 需要执行 $2$ 次及以上的第二操作后才让数列有序，输出 $-1$。

思路分析：

因为数列公差为 $1$，所以交换的数的差和其编号一样都被 $k$ 整除，所以可得到： $\lvert p_i-p_j \rvert\bmod k=0$。

同时，因为输出的结果只和第二次操作次数有关，因此可以设置变量 $cnt$ 表示操作次数。

  1. $cnt=0$ 时，即没有操作过第二操作，输出 $0$。

  2. $cnt=2$ 时，因为两个数字交换时， $cnt$ 会计算两次，故这表示操作过第二操作 $1$ 次，输出 $1$。

  3. $cnt>2$ 时，可以理解为做过 $1$ 次以上的第二操作，输出 $-1$。
