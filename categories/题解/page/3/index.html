做题感想：还是吃了很多没有看清数据范围的亏。

本来以为要二分做，后来发现数据范围小了，可以暴力加前缀和。再后来发现我又忘记了开 long long。

题目思路：

首先带入一个 $x$ 值，对于 $A_i<2\times x$ 的，可以将损失定为 $x$。对于 $A_j >2\times x$ 的，可以将损失定为 $A_j-2\times x$。那么总损失就是所有的两种情况相加了。现在需要求出 $x$ 的值。

对于 $x$ 的值的大小对总损失的影响，就是考虑多一个 $A_i$ 可以用保险解决。那么因此可以得出 $2\times x$ 的值一定和一个 $A_i$ 相等。因此，对 $A$ 进行遍历即可，选择总损失最小的情况。

1- 引子

以前经常和大家介绍为什么小学生，初中生，高中生要学信息学奥赛。

答案只有一个：升学。

必然是为了升学啊，难不成是为了成为技术宅然后拯救世界么。

那今天我来问个问题：

您觉得，这些信息奥赛生，在高中毕业之时，高考之后，大学选什么专业？

要说是考古，我肯定是不信的。

再猜。

计算机？

没错，就是计算机科学系。

按照国内现在的学科分配，大一应该是信息大类，之后，按照成绩分，成绩优秀的去计算机科学系。

简称计科。

计算机科学系的一个主要学习内容，就是算法。

信息奥赛生在高中时期最擅长的，也是算法。

所以对于这些曾经的奥赛生来说，这部分理解起来很简单。

唯一的区别，就是，大学的要求不只是写出程序的实现，更要证明算法复杂度。

这也不是什么难事儿，随便学学就满分了。

然而，算法，只是计算机科学系中教授内容的一部分。

否则，岂不是信息奥赛生可以跳过大学直接工作了么。

这就要提及在信息学奥赛中一个被忽视的部分，也是在计科中非常重要的，堪比算法的内容，那就是：

软件工程。

如果光会算法，完全不会软件工程，或者听了软件工程就恶心，那我保证你连个大厂实习都找不到。

这是个事实，不需要争辩：

1- 如果你是个正经的奥赛生，应该拿个提一没问题

3- 如果拿了提一，你应该能上个一本的计科也没问题

3- 如果你上了个正经的计科，没道理你没学过软件工程

4- 如果你在大学学了软件工程，却完全没学会，说明你挂科了

5- 如果你挂科了，你肯定过不了大厂的人力资源部门的秘书小姐姐的筛选

6- 1到5好好走几遍，看看这个逻辑是不是缜密。

所以软件工程就是这么的重要。

说不可或缺，也没有什么不妥的。

软件工程里面有个重要的方法论，那就是面向对象编程，英文是Object Oriented Programming，简称OOP。

从开始带信息学奥赛这一天算起，我就一直觉得，信息学奥赛，是可以用计算机科学中的软件工程来化解难度的。

只不过，提高组通常与软件工程无关，纯拼算法。

只是，今年的提高组，偏偏就来了这么一道题大模拟，主题就是OOP。

[CSP-S 2023] 结构体 题解

在上纽（上海纽约大学）学了 python 的面向对象编程（OOP），我发现了信息竞赛题目（尤其是这道题）和 OOP 之间的紧密联系。

在学校的 OOP 相关题目往往都是老师提供程序的类和方法框架，然后我们向其中编写代码来解决问题。例如上学期的 Data Structure，我们的栈、队列、哈希表、二叉树等数据结构都是完全按照 OOP 的方式编写的。

信息竞赛与之不同的是，我们没有已有的框架，于是我们需要仔细分析题目，来决定自己需要定义什么类型、变量以及方法，然后再编写具体的代码来解决问题。这样做题思路会比上来就开始写代码要清晰很多。以及信息竞赛使用的语言是不足够 OOP 的 C++ 语言，而且处理的任务很小，目标也是代码效率以及编程效率至上，故很多时候使用较为简单的 struct 以及全局函数即可。

P9761 [ROIR 2021 Day 1] 两台机器 题解
2023-11-06 - /087_P9761 [ROIR 2021 Day 1] 两台机器 题解/

思路分析

给定 $2$ 个机器的启动时间和效率，求在 $k$ 分钟内最大生产数量。

为了让生产数量最大，肯定要最开始启动一个机器，后面在启动一个机器，那么无非就两种情况：先启动一号机器，先启动二号机器。

先启动一号机器收益为：$(k-a)\times x+(k-a-b)\times y$。

先启动二号机器收益为：$(k-a-b)\times x+(k-b)\times y$。

还有一种情况是给定时间不够启动机器，所以要进行判断。

那么综合起来就是：

$$\max(\max((k-a),0)\times x+\max((k-a-b),0)\times y,\max((k-a-b),0)\times x+\max((k-b),0)\times y)$$

注意：这道题题面写的很清楚要开 $\texttt{long long}$ 但是我最开始忘记开了。
