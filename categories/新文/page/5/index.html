信息奥赛生入门的黄金期 新文
2023-10-20 - /098_信息奥赛生入门的黄金期/

转载小周老师文章源自

因为小周原文有敏感字，所以做了一些处理


是信奥的机构，就好好培训学生。

偶尔说个听着有用的，譬如说，“奥赛准备黄金期”，仔细一看主语还都特别混乱：

奥赛和黄金？

期是什么？？

黄金期货？？？

白瞎了“黄金期”这三个字。

今天小周老师就来说说这个“黄金期”。

首先，主语要清晰，“黄金期”，是个主语，前面是要有定语的。这个定语，至少要是“信息奥赛生”。

连在一起，应该是，“信息奥赛生的黄金期”。

但这看起来仍不清晰：

为什么信息奥赛生会有个黄金期？这个黄金期指的是？

所以还需要扩充这个定语变成形容词短语。

信息奥赛生入门的，黄金期。

这看起来就对了：对于打算入门的信息奥赛生来说，有个相关的概念，叫做黄金期。

要想弄明白黄金期，那就要了解在信息奥赛的路上的两件头等大事。

**第一是要早。

第二是要熬。**

所谓的早，就是，如果想要学信奥出成绩，务必要在四年级的暑假开始。

当然二年级也行，这要分教练，也分学生。

对于教练，不管二年级还是初三，能够驾驭语言这是个最基本的要求，光吹牛讲黑话是不够的，要有本事。

对于学生，这就有早慧与否的差异了。但早慧与否，只和低年级有关。

**所以，教练需要的，是能让学生学大学的计算机科学的内容，还学得不烦的能力。

学生需要的，是中文听说读写的能力。**

请注意，不是数学能力，也不是计算机的兴趣，而是中文的听说读写的能力。

那么，我们就把从小学二年级开始的信奥生入门需求，列一列：

二年级的小朋友：
1- 不需要通过初中联赛（CSP-J）的初赛，还太早。他们也根本竞争不过早培生或最普通的初中生。
2- 他们只需要能够让C++跑起来，这就是最直接了当的学习回馈。
3- 虽然小，但比较基本的奥赛编程题在上手之后，也是完全可以做的。
4- 要能看懂题，要能听懂讲解。

三年级的小同学：
1- 仍不把通过CSP-J的初赛作为首要目标，但该学的初赛知识（提高组暨高中联赛的知识），只要能学，就尽量储备。
2- 这个年龄，不管是逻辑能力还是语言理解能力，都有长足进步，所以可以不满足于简单的编程语言本身（在小周老师看来，C++本身，就是简单的）。
3- 不管是不是准备好了，都要开始有对比较复杂的事物的理解能力。
4- 只要能学思维训练（就是奥数啦奥数），就能学信奥
5- 需要有能够在online的信息奥赛中（理解成为没有初赛的初中联赛等级的比赛）拿到奖的能力

四年级的同学：
1- 上述所有要求的客观条件都已经具备
2- 暑假之后就是五年级，从小升初的准备时间上，是最后的练兵机会。
3- 六年级开学的秋天，就必须要拿奖，所以新五的比赛，就和真正的战场没有区别
4- 必须尽全力冲击初中联赛的初赛，就算过不了初赛，也要看看自己的水平差距有多少，在下一年才可以查漏补缺。

五年级的同学：
1- 暑假后开学六年级
2- 暑假后开学六年级
3- 暑假后开学六年级
4- 重要的事情上面已经说了三遍了。暑假不全力拼搏，开学可不是和小学生们抢初中联赛的奖，是和初三生。
5- 恐怖么？还觉得五年级小么？

初中同学（新初一新初二新初三）：

1- 这已经是个全新的时代，锁定的目标是中考；面对的挑战是普一和提一
2- 已经错过最佳入场期了
3- 初中生的分析，在本篇文章中不做讨论。

所以，很显然，这几个年级里面，是存在所谓的信息奥赛生入门的黄金期的。

而这个黄金期，从时间线上来看，有个非常清晰的关键时间节点：

四年级的暑假。

…

这，叫做准备小升初的科普文。

不过，今天小周老师也不是来给文章打样儿的，别家机构愿意怎么写公众号，都是自己的事儿。

只是，什么事儿都别白忙活，别弄了半天，吹了半天牛，说什么百万人群选择，单人指导成千上万奥赛生，结果一看浏览量，30。

如果（更糟的情况），自己娃，不但是普通娃，而且特别不省心，还没有优秀教练带，那基本上，就是……

完蛋一条路。

唯一解，就是找个好教练。

当然，宇宙之大，无奇不有，也有可能会是下面的这种情况：

第一家长不是计算机科学系毕业的，第二家长对算法复杂度还深度感兴趣，第三家长觉得自己下场子教效果好，那么，我推荐这本书：

（信息学奥赛一本通图片）

福建名师董老师的信奥手册，基本上信奥生人人一本。

绝对不忽悠。就问全天下的提高组奥赛生，有谁没有这本书吧？

都有。

所以请家长照着自学，然后教娃。

啥？看了太多的结点现在连结点的“结”都感觉不认识了？

不要吧？刚才不还说对算法有深度兴趣的嘛？

所以，让孩子自学？

您都是成人了，您都学不会，普娃自学能会？

有句老话，兵熊熊一个，将熊熊一窝。

套用在教育领域，那就是，教书的人不行，那学生保证行不了。

这就是为什么大家总想要去好的学校，不是因为好的学校桌椅新。

是因为老师好。

所以家长想跨过教练，指导孩子的，请先行品尝一下信息奥赛一本通。

真不贵，去买一本，值。

所以，”信息奥赛生，要早做规划“的”早“字这件事，已经说完了。

第二是要熬。 这个其实也比较好理解，因为初中联赛就是个起点。高中联赛，才是竞技的舞台。

之所以初赛对于小学生有很大的杀伤力，是因为初赛里考的，都是高中联赛的知识。

只不过，初赛的形式看起来是选择题，人畜无害的。

平时蒙对了俩仨的，家长还能觉得自己家娃天赋异禀，除了每年被初赛结果暴击一次之外，娃和家长，都没有矛盾。

这就属于双赢。

总之，当一个奥赛生闯过了初赛关，拿到了普及组一等奖（普一，也就是初中联赛一等奖）之后，下一个目标，就是提一。

提高组一等奖的水平，大概是全北京市，或者一个省，的前130名。

高中初中小学全都算上。

到了提一之后，就要开始熬了。

什么时候能把最前面的大佬们熬到大学，什么时候自己就能有机会了。

但这个“熬”字，从根本上，还是“早”字的延续。

没有“早”，拿到提一的时候，很有可能就已经高二了，基本上没有时间熬别人。

所以，归根到底就是，学信奥，就是现在。

这就是信息奥赛生入门的黄金期，的隐藏意义。

时光荏苒，错过了小学四年级的暑假，可真是留不了级的。

**如果您家的信息奥赛生是初三，那么这篇文章绝不能错过。

如果您家的信息奥赛生是初二，那么文章里面说的事情，就是明年。

如果您家的信息奥塞生是四年级，那这篇文章一样不能错过：再过两年就是小升初，如果不能进早培、元培、十一2+4，那再数三年，就是这个时刻。**

但是如果，您家的奥赛生已经出于各种原因停留在普及组的入门阶段，其未来或者现在的提高组奥赛生的身份，已经进入销毁期、休眠期、自毁期、叛逆期，不一而足，那这篇文章还真不适合您，直接关了比较省时间。

为什么？

因为现在要说的特长生群体，是普及组和提高组的贯穿类同学。可不是简单的普及组会写两句三脚猫的程序的这个能力而已。

而了解北京奥赛圈子的家长，应该知道，能，以及愿意把奥赛生，从零基础历经完全不摸门的新手阶段，培养到普及组，最终一直训练到提高组（S组）一等奖的教练，极少。

小周老师就是其中一个。

虽然培养出来的奥赛生各区很多名校都有，但自身，不隶属于任何学校。

这就直接能证明本篇文章的中立性。

好，进入正文，下面要说几件事情：

**1- 中考科技特长生，往年报考流程
2- 2022几个名校的报考流程（人大附中，首师附，等）
3- 2022年，各个名校的科特生要求？（重点中的重点！）

1- 中考科技特长生，往年报考流程**

所谓百炼方可成钢，这话一点儿不假。

暖洋洋的五月，对于奥赛生来说，就是战场：因为，到底是不是信息奥赛生里面的钢，就要决出高下了。

不管自己是不是钢，都无法改变名校的录取原则：

名校，只要钢。

普通材质的不收。

在具体操作层面，中考前，科技类初三生要参加学校的科技类校考。如果是信息奥赛生，那就相当于一次省赛等级的比赛。

请注意，不是所有的科技特长生都是信息学奥赛生，不过，只有信息学奥赛生的信息奥赛成绩，是最过硬，也最容易测试的。

至于到时候校考，是初中联赛这样的省赛，还是高中联赛的省赛，这分学校。

这里就不具体比较各校的考题了，一句话就是：

**如果自己水平弱一点，就别扎堆考人大附中。

考不上的。**

如果你连提高组的课程就没怎么好好上，那根本就不要考虑，去了也有可能爆零。

不管报什么学校，请注意，报考科技特长生的高中，是排他性的：意思就是，如果你报了首师附，就不能报人大附。

从时间安排上，各个学校在往年也都是同一时间进行。

所以请一定要慎重选择。

如果报考了适合自己的学校，在考试中发挥优秀，通过了校考，那么就会拥有科技特长生的身份。

不过，有科技特长生的身份，和能够走科技特长通道中考被录取是两件事。

在通过校考的奥赛生中，择优录取。

2- 2022几个重点中学的报考流程

今年和往年流程上没有什么不同。

目前出了招生简章的，海淀区比较集中：有人大附中、首师附这两个市重点（等同于其他省的省重点）；以及两个区重点：二十中和八一学校。

流程上，分两种，一种是市重点的流程，这个要进行严格校考，说得是人大附中和首师附；还有一种，是区重点的流程，这就比较松散，一般是提交材料，比如八一学校。

在报名操作上，无非就是在某一个日期前报名填表，提交简历或者视频。

但无论什么样的流程，都请谨记，您家的奥赛生，只能报一所学校。

所以请谨慎选择。

至于选哪一所考，请看下一个小节。
