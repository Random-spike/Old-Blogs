（一） 拿了北大录取通知书的信奥生

这几天，今年高中毕业的同学应该陆续收到录取通知书了。

于是这天上午，有个我以前带的奥赛生小朋友，给我发了他刚收到的录取通知书。

北大的。

北大，真棒！

对不对？

只是，北京大学专业多了，到底学啥的？

这些专业里面可有些冷门的，论语言，有泰语、菲律宾语、乌尔都语；；论艺术的文学加工，有艺术史论、戏剧影视文学、文化产业管理；论财会，有光华管理学院的会计学；论设计城市，有城市与环境学院的城市规划和城乡规划，当然，要论理工，还得是数学、物理、化学以及计算机科学与技术。

这里面可不是都一个分数，互相可差多了。

比如，北大计算机系，2023年在北京市本科批次的录取人数是10人。

所以这位小朋友被什么系录取了？

既然这是个聊信奥的公众号，显然这张录取通知书应该是计算机科学系的。

那是不是呢？

如果看不清楚是什么专业的话，我再发一张大点儿的。

【关于这位小朋友的个人信息就一概都抹去，就留个姓】

（二）所以这位小贾同学是谁？

小贾是人大附中高中的应届毕业生，以下两件事很重要：

1- 初中不是早培，甚至不是人本，但最终信息奥赛的竞赛成绩一骑绝尘，让人大附中联合总校中的一众尖子连影子都看不到；

2- 高中班不是人大附中传说中的最强早培实验班，然而高考分数也远超相当一部分早培，考了693分，名列北京的前251。

看起来，这是一个别人家的孩子，走了一条别人家的孩子才能走的超学霸的路。是不是？

错了。

那么，如果我说，这个孩子当时的手牌是这样的：

第一，不是六小强的学生

第二，小升初的暑假才学信奥

第三，新初一的信奥，普及组二等奖开局

您会怎么想呢？

用这么三张牌开局的话，那您想不想知道，这是怎么一路走高的？

如果用一句话问的话，那就是：

普二开局的奥赛选手，是怎么从普二，到普一，再一路拼到提高组一等奖以保送人大附中，进而进军国赛拿到NOI银牌，最终考进北京大学计算机科学系？

确切来说，是以下的几个问题：

0- 什么资质的娃普二+开局（普二或者普一）？

1- 普二开局的奥赛选手，怎么继续提升自己？

2- 普二之后是什么？

3- 普一是不是终点？

4- 怎么拼到提高组一等奖？

5- 保送人大附中是什么要求？

6- 国赛资格怎么拿？

7- 怎么考北京大学计算机科学系？

0- 什么资质的娃普二+开局（普二或者普一）？

这要说回到差不多7年以前的海淀黄庄。当时和一个挺小的、不算太知名的数学机构合作了一下，我挑了6个小升初的学生到我的信息奥赛队。

小贾就是其中之一。

这六个孩子里面，小升初学校听起来最大牌的，也就是清华附中的一个分校了。

这六个学生里面后来出了4个信奥高中联赛一等奖。

当然这都是后话。

我刚才也说了，小贾初中不是早培，不是人本，甚至不是任何的六小强。

但有一点请清晰，这六个孩子，都是奥数生，没有一个是没有奥数训练之后的思维基础，以及习惯努力的学习习惯基础的素人。

小贾和其他几个同学一样，奥数不错，人聪明，而且相当努力。

这样一个孩子，在暑假坚持一个月100小时的训练，能够零基础到通过初赛以及复赛拿奖，本身就是巨大的成功。

虽然因为微小失误而拿了初中联赛二等奖，但初中联赛一等奖的潜力，是有的。

所以小贾同学，是个新初一开局的范例。

所以，什么样的潜质能够普二+开局？

那就是：

0- 要有名师，在短时间内从零基础到奥赛能力成型，同时不过度消耗奥赛生体力脑力

1- 年龄够，几率会大（新初一或者新六）

2- 努力程度要够（要有奥数训练的基础）

3- 要按照教练的要求做

这里还要强调的就是，如果年龄不够，不是拿不到普二+（小周老师教的奥赛生，只要过了初赛100%普二＋），而是非常可能闯不过初赛（这个初赛的问题，以后做专题和大家讨论）。

之所以在初中联赛这个阶段要有努力程度的要求，就是因为要过初赛。这件事和数竞是完全不一样的。数竞的初中联赛就已经有智商筛了，而同等级别的信息学奥赛，有的，只是努力筛。

这也不是因为信息奥赛容易，数学奥赛难，而是卷信息奥赛的人，远没有卷小学奥数然后升到初中继续卷数竞的人多。

1- 普二开局的奥赛选手，怎么继续提升自己？

首先，不要妄自菲薄。

有时候普二的成绩，和普一，就差10分到30分，这就是3个测试点而已。如果是这种情况，可以忽视二等奖和一等奖的区别，直接视为自己为普一，继续往提高组努力。

随后，一定要好好分析周边资源。

其中，最重要的资源，就是教练的提高组加工能力。

这里要划一下重点：

提高组。

也就是高中联赛。

普二同学要想提升自己，必须要有提高组的教练，在短时间内把自己带到高中联赛的水平。

这里说的教练分为两种，一种是校内教练；一种是校外教练。

校内教练如果不是名校，那么基本上都没有提高组的成绩，或者尚欠缺带高中联赛的能力。这种事情，不可强求。没有校内高中联赛的教练，那就是没有，不是每个学校都能找到NOI金牌或者IOI金牌作为主教练的。

这就要靠校外教练的辅助。

校外教练曾经的标配是普及组，一般机构只要拿到奖就发喜报，要求也不高。教练水平参差不齐，想找，肯定能找到。

只是，如果孩子已经是普二了，那再学普及组的知识，明显就是浪费时间。那怎么办？

很简单，必须要找提高组的教练。

所以，

校外教练现在的标配，是提高组。

这就不太好找了。

尤其是很多机构在提高组这点上总是支支吾吾，说不清楚。广告里说有提高组的教练，有什么北大清华拿过高联一等奖的教练，最后教课的教练满不是这么回事儿。

但这种机构有个好处，就是便宜。

这就有点像鬼市淘古董，碰到了就是赚到。

前提是鬼市，他真有古董。

但这个机构有没有提高组的教练，您不知道。

最后，校外教练，还存在一个顶配。

这就是能培养出初中生高中联赛一等奖的，名师。

请注意，这里说的是，初中生里面，培养出高中联赛一等奖。

能做到这件事儿的教练，非常非常少。

显然，小周老师是一个。

如我所说，提高组校外名师所负责的，就是初中阶段培养出提高组一等奖的孩子。至于之后去什么学校，都看学生自己的意愿。

在这个层面上，这种校外名师，都是雇佣兵。也就是，家长交钱，教练教课。没有什么藏着掖着不肯教学生这种事。

正常来说，只要是交钱了，那没有任何不教提高组的可能。

除非，是教练自己不会，滥竽充数。

尤其是，教练的能力，一年才有一次检验的机会，所以最好是发现问题，立刻解决问题。该换机构，马上换机构。

但这件事儿是这样，你问教练你会不会提高组，他肯定说会。那怎么判断？

简单，下面两条可以作为判断依据

1- 有没有初中学生曾经在提高组获得一等奖的战绩 （至少提高组二等奖）

有提一，有提二，还必须要确认，这是不是亲徒弟，是从什么阶段教的，普一提一最好一起一个教练带的，如果不能一起，那必须至少要提高组，从头到尾是亲带的才行。那种小学三年级送来听了一次讲座给了名字，之后人家进了省队然后去碰瓷的教练，真心不算。。。

举个例子：

比如说小周老师的提一，全部都是从零基础带出来的。

这种就行。

2- 暂时没有提高组战绩，但宣称能够带出提高组的成绩。带出提高组的成绩。

这，口说无凭的，就只能看教练自身的教育背景了。

这种教育背景，包括但不限于学校的排名、自身的业内声望、所著书籍、学生时期获得奖项，等等。

从写书的角度，福建写了《信息奥赛一本通》的董老师，显然就行；但董老师提高组一等奖一大把，NOI金牌都一大把，所以不是个好的例子。

再比如，最近写了一些初赛的书籍的教练，也可以。

如果教练不肯说什么学校毕业的，那显然不行。

如果教练说了什么学校，但一查是个普通一本，甚至是二本，那一般不行。

这种，其实有很低的概率，能行的，但是机会成本太高，和潘家园买青花瓷一样，不建议冒险。

其实最简单的论据就是，这都2024年了，如果能打竞赛，能教竞赛，怎么还能上不了个985呢？

就千万别信潘家园能淘到仰韶文化的玉龙，都扯淡的。

如果教练是个985，但不是计算机科学系，请避坑，大几率不行。

要知道，信息奥赛这种东西，尤其是职业教练这个层级，我们可以肯定地说，无法速成，都是青少年时代的童子功，当时打了奥赛或者学了算法，就有基础，当时没学没打那就没有。

比如小周老师，虽然在高中时候各种惫懒，但在法国考上了排名第一的巴黎第六大学计算机科学系之后，和法国同学们一起，各种被老师安排打校内奥赛，加上算法专业人工智能专业的各种算法课的锤炼加持，造就了算法天赋的奥赛基因，这是无法伪造和后天补课的。

2- 普二之后是什么？

普二之后，显然是普一。

也就是初中联赛一等奖，或者说，CSP-J的一等奖。

对于没有拿过普一的同学，这个普一是名校校队的敲门砖，是必须要拿的。

所以进入国赛的同学，是非常非常少见在普及组没拿过普一的。

所以，普二之后，最重要的事情，就是第一时间，拿一个普及组一等奖。

其实，普及组一等奖和提高组二等奖，是差不多的难度。

这就又回到提高组教练这件事儿了。

一言以蔽之，普一，这在信奥的世界里面，是个必须的勇者之证，想往上走，必须有。

没有商量。

3- 普一是不是终点？

普一当然不是终点。

不管什么时候拿到普一，不管是小学，初中，还是高中，下一个阶段一定是提高组，也就是高中联赛。

（其实高中生也可以来冲击一下普及组一等奖，也就是CSP-J，这算省赛）

普一算是高中联赛的一个比较不错的起点。

一般来说，信奥圈的教练都认为普一的价值，大于提三（高中联赛三等奖）。

那么，通过普一，再进军高中联赛的学习，也是非常正确的。

我们以小贾同学来举例：

小贾同学第一年拿了普二，第二年拿了普一，第三年就直接冲击提高组了，拿了提高组的一等奖。

这个提升，就非常顺。

4- 怎么拼到提高组一等奖？

这是个非常大的题目，我们不能展开说，绝对写不完。

只能说，这取决于选手的天赋，教练的能力，以及整个奥赛队的气氛。

说到奥赛队的气氛，我们就不得不提一下信息奥赛传统校的初中队和高中队。

一般来说，如果不是北大附中、首师附这种从初中校队往高中校队选拔高中联赛人才的重型信息奥赛强校，普通的初中校队，都是比较松散的。

也就是说，学校对初中校队的期待，其实都不是高中联赛一等奖。

而是不倾注太多精力的同时，能找到天赋型的选手。

这里说的天赋型选手，邓明扬绝对算一个，许庭强是当仁不让的另一个。

于是，初中校队，一般来说，从训练的角度，也就不会特别严格。

前几年我接触的一些初中校队的学生，奥赛能力都比较一般，而且普遍有背诵题解的毛病。

这就是见不到高中校队主教练的初中校队同学的通病。

换一个角度来说，那就是，如果想要获得校内的高中校队资源，就必须要能在高中前，提前获得提高组一等奖。意思是，依靠校外教练的加工能力。

再说回小贾同学。

我已经说了小贾同学的初中完全不是任何六小强，所以初中校队根本都不存在。

全部成绩，包括初联一二等奖，高中联赛一等奖，都是在校外教练小周老师的奥赛队中获得的。

古人说，

【祸兮福所倚，福兮祸所伏】

意思是，主动和被动的因素，其实都是互相依存的。

也就是，如果身处一个周围的同学都在抄题解的初中校队，还不如不参加校队。

5- 保送人大附中是什么要求？

很简单。

需要高中联赛一等奖。

我不确定我表达清楚了没有。

要不然我来个大点儿的？

需要高中联赛一等奖。

这只是一个基本要求。

最最重要的，就是能顺利通过人大附中校考。

这里说的校考，是科技特长生的计算机奥赛校考，不是数学物理。

当时小贾同学参加人大附中校考的时候，题目难度是国赛难度。

做完了之后马上出分，在场教练表示非常满意。

然后小贾同学保送人大附中高中。

所以没有高联一等奖行么？

不行。

不如报101。

如果想要在初中阶段得到这种能力，那就只有一个可能：

找个顶配的高中联赛教练。

倒不一定是我哈，您能找到哪个，就是哪个。

反正小贾同学碰到的是我。

6- 国赛资格怎么拿？

如果不在信息奥赛的强校，就别惦记国赛了。

因为，如果不是天赋异禀，直接北京市排名（省排名）在前十五，那就没可能。

同时，如果没有人大附中、北大附中、首师附、十一、八十、北师大实验的校队加持，那就算是天赋异禀， 也够呛能排到前十五。

名校校队的训练深度（国赛水平选手的互相促进），训练广度（和外省名校联合训练），是学生个体完全无法达到的。

放眼全国，基本上所有国赛金银牌，都是和自身学校的信息学奥赛的高中校队息息相关的。

比如说，小贾同学就和我说过，当年NOI国赛的时候，和IOI金牌许庭强一个宿舍，同宿舍的还有八十中的省队成员，晚上讨论一道题，大家就都和许庭强学了思路，结果考场上考了个类似的，大家用了几天前许庭强讲的思路，得分点丢分点都大差不差。

这就是国赛选手的环境。

这部分的成绩，比如说小贾同学的国银，就完全是学校的国赛加工能力，没有校外教练的功劳。

如果说有，也就是以前打的提高组基础。

为什么呢？因为这个部分的比赛，校外教练一般不涉及。

不是教不了，而是学生群体不够大，投入产出比太小。

简单来说就是，不划算。

那么，怎么才能去国赛？

那你要先去个信息奥赛的名校，不然可能性无限接近于零。

怎么去信奥传统名校？

当然是初中拿到提高组一等奖了。

别无他法。

7- 怎么考北京大学计算机科学系？

这个么，我提供两个思路：

1- 拿国赛金牌

有个原来和小贾同在我的奥赛队的小陈同学，在跟我拿到普一和提一（初二）之后，NOI国赛金牌拿了两次。

这种国家集训队金牌，拿一次就保送了，根本不用拿两次。

拿两次金牌的小陈同学，纯粹就是挑战自我

2- 拿国赛等级的银牌+高考高分

这里的国赛等级，包括但不限于NOI，NOI冬令营，APIO（亚洲及太平洋信息学奥赛）的银牌+。

有些家长可能不理解，如果高考高分，为什么还需要国赛银牌？

原因很简单，想考北大，和想考北大计算机系，这可完全是两个概念。

两个问题：

a） 分数过线，就能上计算机系么？

b） 分数过多少，能上计算机系？

这里，有没有一个国赛级别的银牌+，就相当重要了。

所以究根到底，小贾同学的成功之路，还是因为初中拿了提高组的一等奖。

结语：

奥赛生，其实不用瞄向国金国银，这个标准不适用于所有人。

要说厚积薄发这句话，还真不是当代的词，这是出自苏轼的《稼说·送张琥》。

他说：【博观而约取，厚积而薄发，吾告子止于此矣】

意思是说，博览群书，去芜存菁，重视积累，此所谓厚积，最终谦虚谨慎，缓缓而行，宁缺毋滥，正所谓薄发也。

至于厚积薄发之后，是否能一鸣惊人，这要看命。不管是不是一鸣惊人，前面都是埋头苦干了挺久。

而且满打满算，也就是两年而已。

在这么少学习时间的前提下，还基本上是自学，就在阿里巴巴数学竞赛中压过了一众从小学就开始搞竞赛的竞赛生成长起来的各个学校的数学尖子，包括但不限于清华、北大、麻省、剑桥、普林斯顿。

前情提要

lxm老师在上公开课的时候，毫不犹豫的指出，新时代中国特色社会主义现代化建设者nitianyi的名字非常的好！而旧时代清朝特色封建统治闭关锁国战略策划者quansiyuan的名字非常的差！

为什么呢？

她认为，nitianyi的名字读音比较好，而quansiyuan的读音不好。（应该是在上一节关于古诗格律分析的一节课）

靠谱的朋友

我从一个比较可靠的朋友那里专业的得知。nitianyi名字三个字分别是阳平，阴平，去声。而quansiyuan的名字三个字分别是阳平，阴平，阳平。一眼丁真为阳气过重。厌氧建议关闭O2

一个比较可靠的朋友这么评价。

按照这三个字的声调，我们可以判断“nitianyi”的平仄为“平平仄”。

总结来说，“nitianyi”这个名字的平仄是“平平仄”，其中“ni”和“tian”都是平声，而“yi”是仄声。这样的平仄组合在汉语中具有一定的音韵美感，读起来和谐流畅。

看起来这位朋友并不那么靠谱，说了一些废话。

这个比较可靠的朋友继续说

关于为什么不具有美感，有以下几个原因：

缺乏变化：在中文诗歌和韵律中，平仄的交替变化是重要的美学元素之一。这种变化可以产生节奏感和音乐性。然而，“quansiyuan”这个名字全是平声，缺乏起伏和变化，因此听起来较为单调，不够悦耳。
重复过多：连续三个平声字的使用，使得名字在听觉上缺乏层次感和变化，显得过于平淡。在命名时，适当的平仄搭配可以增加名字的节奏感和韵律美。
文化背景：虽然现代对于名字的审美已经与古代有所不同，但很多人仍然倾向于选择具有一定韵律感和美感的名字。在古代文化中，平仄的和谐搭配被视为名字美感的重要组成部分。因此，即使在现代，很多人仍然认为具有平仄变化的名字更具美感。
综上所述，“quansiyuan”这个名字在平仄上缺乏变化和起伏，因此听起来较为单调，不够悦耳。这种缺乏美感的原因主要源于平仄的重复过多和缺乏变化。当然，名字的美感是一个主观的评价标准，不同的人可能会有不同的看法。

这么说来，quansiyuan这个名字同样因为平声太多而不如nitianyi这个名字了。

奇怪的现象

但是呢？我又举了一些例子。

在20世纪某年代的名字中，很多都是平平平或者平平的。

比如：

阳平，阳平，阴平。

阴平，阴平，阳平。

阳平，阳平，阳平。

阳平，阳平，阳平。（和上一个人平仄相同但名字不一样）

以及2个字的同样可以举出很多例子。

这些人我们都是熟知且权威的，但是由于某些原因，我不放上来了。

而且名字中全平声的例子在该群体中非常多，看来quansiyuan的名字并不差。

以及一些拓展

某老师经常跟我们讲：「你们的创新点在哪里？」

我认为这个问题非常的好，和nitianyi的名字一样好。

nitianyi的名字还好于：可以进行非常多次“变异”。

为什么怎么说呢，如果把yi改成阳平，

那么nitianyi的名字将会富有意义/xyx，同时又因为读音完全和quansiyuan↑→↑变得相同了（阳平，阴平，阳平）名字又变得非常的差。

看起来lxm老师会不会是为了证明nitianyi不应该读成↑→↑呢？

这时我才发现原来另外一个nitianyi读音也是↑→↑（猜猜ta是谁呢？）
