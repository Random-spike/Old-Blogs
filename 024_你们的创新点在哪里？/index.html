2024-08-20 - /024_你们的创新点在哪里？/

一位老师常说，

你们的创新点在哪里？

这是我引用这句话的0xff次了。我说有这么多就是有这么多。如果你不认为有这么多，那是因为很多文章我隐藏了。

为了让这句话更有意境，我们不妨设这个老师的名字是Luxun。

你们的创新点在哪里？

——Luxun

那我们自然是很有创新点的。

以此文纪我们初中的丰功伟绩。正如某年某区语文一卷所引用人民日报的那篇文章一样。

碑需自立，名由己传

—— 人民日报

2.粉笔  link

很多学生都喜欢玩老师的粉笔，不包括大玮专用粉笔。

粉笔有很多种花样玩，可以用来打二战，可以用来制作烟雾弹，也可以在别人的桌子上施展艺术天赋。

我作为常年占据素质榜 RK 1\text{RK 1}RK 1 的人，自然是 总来不会涉及上述事情的。

我们曾经做过粉笔雕塑。

用针，剪刀，尺子作为工具，用粉笔作为材料，制作一些人像，虽然看起来并不真实，但还是很好玩的，可惜就玩了一天。我制作的 444 个雕塑被人破坏了，倒在地上死无全尸。后来我就不玩了。

粉笔中，彩色粉笔的用处格外的少，因为白色粉笔质量优秀，老师和学生都喜欢使用。又因为黑板擦，扫帚有其他的用途（详见后文：冰球），因此我们班成为教具新陈代谢最快的一个班，每次领取也自然需要拿彩色粉笔，于是乎，彩色粉笔泛滥成灾。

为了解决这个难题，卡洛斯·刘自告奋勇的为班级奉献力量，拿出他的水杯将彩色粉笔研磨成细末，又因为某个同学有一个南瓜灯。我们便把水和粉笔灰倒入其中。

出现了一个自告奋勇的同学就会出现下一个，某同学自愿将咖啡粉倒入其中，经过搅拌，就成为令人垂涎欲滴的饮品了！

制作完这个胶体过后，我们并不满足，由于材料限制，我们加入了某个同学的柠檬，和一些枯萎的树叶。

试想你面前放着一杯由粉笔灰，咖啡粉，柠檬，树叶，氧化氢构成的混合物，你怎能忍住不去尝一口的诱惑呢？

P.S. 某化学老师说过，粉笔是很多工厂制作豆腐的必备原料。(胶体的凝聚)所以粉笔时刻实用的。

但是正当我们举办品茶会的时候，X出现并且制止了我们，也避免了校门口被 120120120 堵住的情况。

尽管最后，卡洛斯·刘用薯片罐子保存了我们的饮品。

突然想到的，我和wyn保存这2024.2，喝了一半的冰糖雪梨，目前仍在我初中的书包柜，不止安好可否？

回到教室中，wcx同学如释重负的用富有深情的眼神看着他失而复得的擦汗巾。一名叫做ssy的同学突然说:

  • 某酒店厕所爆炸

具体内容是某人在酒店上厕所的时候，可能因为积蓄了太多沼气，并且抽烟的火星掉入其中，马桶爆炸了，那个人没有被炸伤，但是一栋楼的马桶都炸开来了，因为管道是联通的。

  • 夫妻大盗偷车成功后深情表白

其实应该并不是夫妻，可能是情侣关系。他们偷取了一辆在地下停车库停着的奥迪，车内行车记录仪显示男性盗车者进行了表白。

巧合的是，他并没有将女性盗车者的名字直接说出，应该只报出了一个子串，这个子串和我们班某个人的名字的子串是全等的，没有哈希冲突。于是我们班的某个人的某什么，也就是卡洛斯·刘一再强调盗车者不是他，他也不会说那句话。

16.电信诈骗  link

社会考察的时候，给每个小组长下发了班级所有同学的通信表，也就是电话号。

德尔塔爆炸 同学作为我们能量最大的原子中的一个电子，开发了给别人打诈骗电话的方式，我们的内容很简单。

  • 冒充保险的客服，但是没有收费渠道。
  • 伪造出别人亲人的消息。

后来，一个叫做wy的人开发出了短信诈骗，他冒充的是楚黎教育，还是用了我的邮箱地址。

后来，我们开始和客服打电话，比如米哈游，尚海整装，联邦调查局。对于米哈游，基本是询问一通原神的问题后，用礼貌/不礼貌的方式得出结论原神降智游戏。

到了今年暑假，也就是最近。

我和其他班的几个同学又因为无聊，开发出新的诈骗。

给别人打电话，然后不说话，听别人说喂？，我们随机拨电话，然后录音录下来。

这是我们听到几个比较搞笑的：

  • 某老太用及其祥和的嗓音询问多次 喂 ,你说话呀

  • 某老太边打电话边跳广场舞，因为我们听到了那欢快，富有节奏感的韵律。

  • 某中年男子没有素质的谩骂，我们几个联合把他骂回去了。我们很文明，用国际社会几个臭名昭著的政治人物骂他。

最后，某同学胆子很大，给化学老师打电话。他打电话的时候，我们都在笑，尽管控制了音量，他小声跟我们说：「等她挂了电话我们听录音」然后听录音的时候发现他的这句被录进去了。

17.叠叠乐  link

有些同学，喜欢卷。我们要阻止他们！

我们有很多办法，比如：

  1. 把他的文件袋和笔袋藏起来。

  2. 把他的笔放在吊灯或者空调上。

  3. 用压路机的物理方式阻止他。

具体的，一个人直接跑过去，叠在他的身上，因为那个很肥大，所以他就没法写字了。由于很快会招来一群好事者，所以会叠很多人，会出现很大的响声，X就会出现并且抓住第一个好事者，也就是倒数第二层。

18.窗帘  link

我们教室的窗帘，有一个吊绳，可以用来控制升降。

我们教室后面，有 333 个挂钩，不知道是干什么用的。

于是我们喜欢摆动吊绳，试图让吊绳同时套在 333 个挂钩上面，这三个挂钩呈倒三角排列。

摆动吊绳搭住一个挂钩很简单。

挂住 222 个需要技巧。

挂住 333 个还是有一定难度的。

大概成功的人不超过 444 个？

有一节道法课，我不想听讲，于是我换位置做到了最后一排，专注的玩窗帘。

总共尝试了一千多次？我也成功了。

19.乒乓球&冰球  link

我们班级教室的门口有一块画了 4×44\times44×4 的格子空地，我们用来打乒乓球。

我们使用乒乓球，但是不使用乒乓拍，用手打，开发了很多模式，比如1v1，2v2,1v1v1v1…

有时候我们会玩气球，在隔壁的空地，找一根水泥的分界线，就开始打气球。

wyh把我们的气球用一指禅搞爆了QAQ。

还是冰球最高级。

因为某些原因，我们很喜欢玩老师的教具，比如黑板擦。

为了看着更像冰球，我们拿来了扫把，通过挥舞扫把的方式驱赶放在地上的黑板擦，然后打入设定好的对方的球门。

值得一提的是，扫把和黑板擦都是新领取的，这也是我们班教具新陈代谢快的原因。

因为打冰球太消耗扫把了，我们学校的扫把质量很差，玩了2天扫把柄就断了，黑板擦也裂开来了。

20.值日生  link

我们要求每一个同学值日，除了劳动委员，因为劳动委员每天都需要帮助同学值日。

值日的有以下项目：

  • 扫地
  • 拖地
  • 擦窗台
  • 擦黑板

擦窗台最轻松，擦黑板最好玩。

某个擦窗台的同学破坏了我的 444 个粉笔雕塑，我不说是谁。

我看用全湿的抹布在黑板从上往下拖，效率很高，创造了有趣的擦黑板方式。

有时候老师看到黑板没擦会叫班长和劳动委员擦。然后X会怪罪到值日生头上，zzq同学经常被怪罪。

可能有趣的事情还有很多，但是我暂时回忆不起来。
