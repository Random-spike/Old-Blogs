猥琐不堪的florr  link

序言：猥琐不堪的florr猥琐不堪，我谢谢你哦。

愚蠢不堪的运气  link

众所周知，我在我们班的运气是众所周知的好，显然，我没有在说反话，按唯物辩证的观点来看，运气好和运气差是相对的。相对性绝定了世界上并没有什么是运气绝对好或者绝对差的，这不仅仅是在合卡的方面体现。这也许可以通过我红蚁穴2红蚂蚁出2红yucca来解释，也可以通过我700红不出青来解释。

事实上，700红不出青是运气极好的行为，这个概率应该低得可怜。

我们需要放平心态，打电动是为了乐趣，乐趣源自于刺激的高波，如果青卡太多高波没意思，更高又上不去，这就很无趣了。打电动技术十分重要，是与装备同等重要的，两者是相乘的关系。所以，我要借我装备奇差无比之机，多多练技术。等我装备好了，由奢入俭难，就不能放下功利之心来提升技术了。所以乘我还在“俭”的阶段，要好好练技术，也许什么时候就能运气爆发了呢。

再次阐述打电动是为了乐趣的观点。在florr我们需要寻找各种各样的卡。比如世界树，我遇到9红金瓢虫仍没有。虽然照我的运气来说至少20个才能出，但我认为这总比青卡好出多了，只要队友有青草，25，26就能遇到一个，这多容易啊。

对于运气炸掉的春菊 @yuzu1207 ，陆菊 @lujushang 以及猥琐不堪需要好人的王禹菊@lorinebluescale，我们不能以妒忌之心看待他们。要认为他们是小菊花才行。这就是孟菊所谓的大丈夫精神:贫贱不能移，富贵不能淫。

电动电动  link

打电动不仅有florr一个游戏，我们可以在yorg搞事情，在paper统一全球，在diep，arras被追着打，在其他小游戏整活，或者在QQ刷屏喊2210,3110@lorinebluescale，3577,3510,35czh@wsyuanlaotai,愉悦身心。

虽然话这么讲，但是我仍然不会放弃florr，明天做好爆肝准备，可能是最后的青卡，至于为什么，你们懂得，下一周和录播局一样惨了。

如果到了下学期复学，我仍然会用discord在学校电脑上电动，但愿加速器给力吧。

加速器  link

说道加速器，我不由得想起我岌岌可危的流量，我一天网课不加节制至少2个G的，而且网课奇卡无比，10个G马上会被我挥霍干净，虽然我可以不停地搞邮箱，但这着实有点烦人啊，对了，如果录播局或者wyh看到了，能不能告诉我这加速器是不是每个月10个G啊，月底刷新吗？
