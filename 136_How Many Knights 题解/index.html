How Many Knights 题解 题解
2023-07-21 - /136_How Many Knights 题解/

原题 How Many Knights 传送门

题目大意：

骑士（马）在象棋中走“日”字攻击，可以攻击到不在同一行或者同一列且曼哈顿距离为 3 的格子。

现在给定一个 M×NM\times NM×N 的平面，求出最多放多少个骑士在这个平面里面。

思路分析：

这是一个规律题，经过推算和尝试后不难发现以下结论。

设置变量 ansansans 为可以放马的最多数量。

ans = \begin{cases} M &\text{if } N=1 \\ N &\text{if } M=1 \\ \lfloor\frac{N}{4}\rfloor\times4+2\times\min(N\bmod4,2) &\text{if } M=2 \\ \lfloor\frac{M}{4}\rfloor\times4+2\times\min(M\bmod4,2) &\text{if } N=2 \end{cases}

其余情况下：

ans=⌊N×M+12⌋ans=\lfloor\frac{N\times M+1}{2}\rfloor ans=⌊​2​​N×M+1​​⌋
