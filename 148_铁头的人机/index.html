铁头的人机 旧文
2023-06-16 - /148_铁头的人机/

世界上头最铁的人，莫非是猥琐人机了  link

铁头人机大战普大吉  link

无人机，头是世界上最铁的，因为他深爱铁头泥，就连天皇都为之折服，所以天皇，猥琐人机，以及菊石，下课都争相坐在铁头泥后面，希望自己有铁头泥一般的超级铁头，虽然天皇权威巨大，菊石善于勾引,猥琐人机还是追到铁头泥，人机拥有超级大铁头。

铁头有什么用呢？

上猥琐蒲大菊的菊花愚蠢音乐课时，人机张口就来：“天皇刘泽奇，傻逼普大吉…”蒲大菊瞬间面部扭曲张牙舞爪屎屁齐喷哇啦哇啦变身猿猴耳朵喷鼻屎得愤怒起来。整个音乐教室便充满普大吉的污秽之物和各种恶心的气体混杂起来，酥肉饼还说：“哇粑粑超人变身！”

人机为了在他的妻子铁头泥面前维持形象，和猥琐蒲大菊争锋相对，变大喊：“皇军不可战胜！你tm哪根葱 马的滚回你的厕所和zms一起咣咣咣咣去，小心我拿铁头铁死你”

普大吉，怒不可遏菊花开裂鼻屎横飞屎屁齐鸣嘴喷狗屎，：“脓ve螚瓦剌瓦剌”

人机：“太逊了，看我铁死你”

说罢，人机带着他的铁头以吃菊花之力，雷霆破坏之势，又有何瑞放屁助阵，以50kg*3000000km/s *1/2 的动能对普大吉进行毁灭性打击，一代传奇普大吉，竟然被人机的铁头撞死！

天皇之怒  link

普大吉死后，人机越来越猥琐，头越来越贴，越来越爱铁头泥，每节下课都要依偎在铁头泥的身上或者坐在她的椅子上，十分猥琐。遭到了天皇和菊石的仇恨。

天皇乃是大日本帝国最伟大的天皇，菊石乃是大日本帝国最煞笔的天皇，加上日本得力干将录播局，卢波局，播剧，波菊，伯菊，以及春秋二菊的相助下，要和人机抢夺铁头泥。

天皇率领皇军，浩浩荡荡的来抢夺铁头泥，人机十分愤怒，铁头泥十分担忧。人机再次使用铁头把皇军横扫。铁头泥十分欣喜。

铁头之暴毙  link

铁头泥为什么会突然暴毙呢？

这得从shk调戏铁头泥说起。

shk十分猥琐，爱国10,7,3,12等等人，还在黑板上写“我喜欢10！”被老姜菊发现后还好不羞涩，毫不掩盖自己对10的爱。果不其然，他现在又爱死了铁头泥。

shk一下课，总是含情脉脉的看着铁头泥和无人机卿卿我我打情骂俏无所不聊，又一脸嫉妒的看着他们，上课时，shk总试图调戏ttn，还曾经给铁头泥身上泼水，下课后被无人机带着自己的铁头把shk一顿暴打。shk干最猥琐的事便是掀开铁头泥的衣服，铁头泥亲自暴打苏恒康。

苏恒康为了更好的调戏铁头泥，变和人机大战，苏恒康动用猥琐四人帮，袁老太，普大吉，田车轮，面子哥的力量，和人机展开殊死搏斗，不料误杀铁头泥，人机哭天喊地。

惋惜铁头泥的去世。
